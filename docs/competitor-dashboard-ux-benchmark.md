# 인테리어코치 대시보드 v2 고도화 — 경쟁사 대시보드 UX 벤치마크 리서치

---

## 1. 서비스별 대시보드 분석

---

### 1.1 Buildertrend

**개요:** 미국 최대 건설 프로젝트 관리 SaaS. 주로 주택 건설업체(Home Builder), 리모델러, 전문 시공사 대상. 2006년 설립, 1백만 이상의 프로젝트 관리 실적.

**메인 KPI 카드 구성:**
- **To-Do 요약** — 오늘/이번 주 미완료 태스크 수
- **진행 중 프로젝트 수** — 전체 활성 프로젝트 카운트
- **미결 변경주문(Change Orders)** — 승인 대기 건수 및 금액
- **미수금(Outstanding Receivables)** — 청구 완료 / 미수금 총액
- **일정 요약** — 오늘/이번 주 주요 일정 리스트

배치 순서: 재무 지표(미수금, 변경주문) > 프로젝트 현황 > 일정/태스크 순으로, 경영자 관점의 "돈 먼저" 배치 철학이 뚜렷함.

**차트/시각화:**
- **프로젝트 진행률 바 차트** — 개별 프로젝트별 % 완료 수평 바
- **캐시플로우 라인 차트** — 월별 수입/지출 추이 (선택 기간 필터)
- **일정 간트 차트** — 프로젝트별 타임라인 시각화 (별도 화면이나 대시보드에서 미니 버전 제공)
- **예산 vs 실제 비용 비교** — 프로젝트별 예산 대비 실제 지출 비교 바 차트

**네비게이션:**
- 좌측 사이드바 고정형 (아이콘 + 텍스트 라벨)
- 1차 메뉴: Dashboard, Project List, Calendar, Financial, Messages, Reports
- 프로젝트 선택 시 2차 사이드바(서브 네비게이션)가 나타나는 2-depth 구조
- 상단에 글로벌 검색 바 + 알림 벨 + 프로필

**모바일 대응:**
- 전용 네이티브 앱 (iOS/Android) 제공
- 모바일에서는 대시보드가 카드 스택 형태로 세로 재배치
- 현장 사진 촬영/업로드, 일일 로그 기록 등 현장 특화 기능 강조
- 오프라인 모드 일부 지원 (일일 로그 작성 후 동기화)

**개인화/커스터마이징:**
- 대시보드 위젯 순서 변경 가능 (드래그 앤 드롭)
- 프로젝트별 커스텀 필드 추가 가능
- 리포트 생성 시 필터/컬럼 커스터마이징 지원
- 대시보드 레이아웃 자체를 변경하는 수준의 커스터마이징은 제한적

**강점 (인테리어코치 관점):**
- 재무(미수금, 변경주문)를 대시보드 최전면에 배치하여 경영자가 즉시 현금흐름 파악 가능
- 프로젝트 진행률을 한 화면에서 멀티 프로젝트로 비교 가능
- 일일 로그 + 사진 기록 기능이 현장 관리에 매우 실용적

**약점 (인테리어코치 관점):**
- UI가 정보 밀도가 높아 디지털 비숙련 사용자에게 학습 곡선이 가파름
- 한국 인테리어 업계 특유의 업무 플로우(견적서 양식, 세금계산서 연동 등)와 맞지 않음
- 다크 모드 미지원 (라이트 모드 전용)
- 가격대가 높아 소규모 업체에 부담 (월 $99~$399+)

### 1.2 CoConstruct

**개요:** 주택 건축 및 리모델링 전문 SaaS. Buildertrend에 2022년 인수됨. 주로 커스텀 홈 빌더와 리모델러 대상. 견적/선택(Selections) 관리에 특화.

**메인 KPI 카드 구성:**
- **활성 프로젝트 수** — 현재 진행 중인 프로젝트 총 수
- **금주 예정 작업** — 이번 주 스케줄된 작업 항목 수
- **미결 선택 사항(Pending Selections)** — 고객이 아직 결정하지 않은 마감재/옵션 수
- **예산 현황** — 전체 프로젝트의 예산 대비 지출 요약
- **미결 변경주문** — 승인 대기 중인 변경 건수

배치 순서: 프로젝트 현황 > 고객 의사결정 대기 항목 > 재무. "고객과의 소통 상태"를 강조하는 배치.

**차트/시각화:**
- **선택 사항(Selections) 진행 상태** — 카테고리별(바닥재, 타일, 조명 등) 선택 완료/미완료 progress bar
- **예산 파이 차트** — 카테고리별 예산 배분 비율
- **프로젝트 타임라인** — 간소화된 간트 뷰
- **비용 추적 테이블** — 카테고리별 예산/실제/차이 테이블 (차트보다 표 중심)

**네비게이션:**
- 좌측 사이드바 (접기/펼치기 가능)
- 1차 메뉴: Dashboard, Projects, Financials, Communication, Reports
- 프로젝트 내부에서 탭 방식의 서브 네비게이션 (Specs, Selections, Schedule, Budget, Files, Communication)
- 상단 바: 검색, 알림, 프로필

**모바일 대응:**
- 반응형 웹 + 네이티브 앱
- 모바일에서 견적/선택 사항 승인 워크플로우 지원
- 현장 로그 및 사진 업로드 가능
- 대시보드는 모바일에서 단순화된 리스트 뷰로 전환

**개인화/커스터마이징:**
- Selections 템플릿을 업체별로 커스터마이징 가능 (마감재 카테고리, 옵션 목록)
- 견적서 양식 커스터마이징 지원
- 대시보드 위젯 자체의 배치 변경은 제한적 (고정 레이아웃에 가까움)

**강점 (인테리어코치 관점):**
- "Selections" 기능이 인테리어 업계의 마감재/자재 선택 프로세스와 매우 유사 — 고객에게 옵션을 보여주고 선택을 받는 워크플로우
- 고객 소통 중심의 대시보드 설계가 고객 응대가 잦은 인테리어 업계에 적합
- 예산 추적이 카테고리별로 세분화되어 공사비 관리에 유용

**약점 (인테리어코치 관점):**
- 대시보드가 다소 심심함 — 시각적 임팩트가 약하고 정보 밀도가 낮음
- 한국 시장 미지원 (영어 전용, 달러 기반)
- 마케팅/세무 기능 부재 — 순수 프로젝트 관리에 집중
- Buildertrend 인수 후 독립 제품으로서의 방향성 불확실

### 1.3 Houzz Pro

**개요:** Houzz 플랫폼의 비즈니스 관리 도구. 인테리어 디자이너, 리모델러, 건축가 대상. 리드 생성(마케팅) + 프로젝트 관리 + 재무를 통합. B2C 마켓플레이스와 B2B SaaS가 결합된 독특한 모델.

**메인 KPI 카드 구성:**
- **새 리드(Leads)** — 이번 주/월 새로 유입된 잠재 고객 수
- **활성 프로젝트** — 진행 중인 프로젝트 수
- **미결 견적(Open Estimates)** — 발송했으나 아직 승인되지 않은 견적 수
- **수금 현황** — 최근 인보이스 상태 (결제 완료 / 대기 / 연체)
- **리뷰 요약** — 최근 고객 리뷰 평점 및 건수

배치 순서: 리드/마케팅 > 견적 > 프로젝트 > 재무 > 평판. 고객 획득부터 완료까지의 퍼널 순서를 따름.

**차트/시각화:**
- **리드 소스 파이 차트** — Houzz 마켓플레이스 / 직접 유입 / 추천 등 리드 채널별 비율
- **월별 매출 추이 라인 차트** — 최근 6~12개월 매출 트렌드
- **프로젝트 스테이지 파이프라인** — 리드 > 견적 > 계약 > 시공 > 완료 단계별 건수 (깔때기 또는 칸반 스타일)
- **견적 전환율** — 발송 대비 승인 비율 추이

**네비게이션:**
- 좌측 사이드바 (아이콘 중심, 호버 시 라벨 표시)
- 1차 메뉴: Dashboard, Leads, Projects, Estimates, Invoices, Time Tracking, Reports, Houzz Profile
- 상단 바에 검색, 메시지, 알림
- Houzz 마켓플레이스 프로필 관리 페이지로의 연결 동선

**모바일 대응:**
- 전용 네이티브 앱 (iOS/Android)
- 모바일에서 견적서 작성/발송 가능
- 3D 플로어 플랜 뷰어 모바일 지원
- 고객과 앱 내 메시징 가능
- 대시보드 카드 세로 스택 배치

**개인화/커스터마이징:**
- 견적서 템플릿 브랜딩 (로고, 색상, 약관)
- 프로젝트 스테이지 명칭 커스터마이징 가능
- 대시보드 위젯 배치 변경 가능 (일부 플랜에서)
- 3D 플로어 플랜/무드보드 도구 내장

**강점 (인테리어코치 관점):**
- 마케팅(리드) + 프로젝트 관리 + 재무를 하나의 대시보드에 통합 — 인테리어코치의 올인원 컨셉과 가장 유사
- 리드 퍼널 시각화가 영업 관리에 매우 효과적
- 견적서 → 인보이스 → 결제까지 이어지는 seamless 워크플로우
- 3D 시각화 도구가 인테리어 업계 특화 기능으로 차별화

**약점 (인테리어코치 관점):**
- Houzz 마켓플레이스에 종속적 — 독립 SaaS로서의 가치가 약간 제한적
- 한국 시장에서 Houzz의 인지도가 낮음
- 현장 관리(일일 로그, 공정 관리) 기능이 Buildertrend 대비 약함
- 세무/회계 연동이 미국 시스템 기반 (QuickBooks 등)

### 1.4 빌드온 (BuildOn)

**개요:** 한국 건설현장 관리 앱. 종합건설사, 전문건설사, 현장소장 대상. 현장 일보, 공정 관리, 인력/장비 관리에 특화. 모바일 퍼스트 설계.

**메인 KPI 카드 구성:**
- **오늘 현장 인원** — 금일 투입 인력 수 (직종별)
- **공정률** — 전체 공사 진행률 % (대표 수치)
- **금일 작업 현황** — 오늘 예정된 작업 / 완료된 작업 수
- **안전 점검 사항** — 미해결 안전 이슈 수
- **날씨 정보** — 현장 위치 기반 오늘/내일 날씨

배치 순서: 오늘의 현장 상황(인원, 날씨) > 공정 진행 > 안전 > 주간 요약. 전형적인 "현장소장 아침 브리핑" 관점.

**차트/시각화:**
- **공정표(S-curve)** — 계획 대비 실제 공정률 S커브 비교
- **일별 투입 인력 바 차트** — 최근 2주간 일별 투입 인원 추이
- **직종별 인력 파이 차트** — 목공, 전기, 설비, 타일 등 직종 비율
- **안전 점검 현황 리스트** — 체크리스트 완료율

**네비게이션:**
- 하단 탭 바 (모바일 앱 기본) — 홈, 현장, 일보, 일정, 더보기
- 웹 버전에서는 좌측 사이드바
- 현장(프로젝트) 전환은 상단 드롭다운 셀렉터
- 알림은 상단 우측 벨 아이콘

**모바일 대응:**
- 모바일 퍼스트 — 앱이 주력, 웹은 보조
- 현장 사진 촬영 시 위치/시간 자동 태깅
- 일보(일일 보고서) 모바일 작성에 최적화 — 음성 입력 지원
- 푸시 알림으로 현장 상황 실시간 전달

**개인화/커스터마이징:**
- 공종(작업 유형) 리스트 커스터마이징
- 일보 양식 커스터마이징 (항목 추가/삭제)
- 대시보드 레이아웃 변경은 미지원 (고정형)
- 안전 점검 체크리스트 항목 커스터마이징

**강점 (인테리어코치 관점):**
- 한국 건설 현장의 실무 프로세스를 정확히 반영 (일보, 인력 관리, 안전 점검)
- 모바일 퍼스트 설계가 현장 이동이 잦은 인테리어 업체에 적합한 참고 모델
- 현장 사진 + 위치 태깅이 시공 기록/증빙에 유용
- 날씨 정보 연동이 외부 공사 일정 관리에 실용적

**약점 (인테리어코치 관점):**
- 대규모 건설 현장 중심 — 소규모 인테리어 현장에는 기능이 과잉
- 견적, 마케팅, 세무 등 경영 관리 기능 부재
- UI가 기능 중심으로 시각적 세련미가 부족
- 다크 모드 미지원

### 1.5 현장의신

**개요:** 한국 인테리어/건설 현장 관리 앱. 중소 시공업체, 인테리어 업체 대상. 시공 일지, 하자 관리, 고객 커뮤니케이션에 특화. 비교적 최근 출시되어 모던한 UI.

**메인 KPI 카드 구성:**
- **진행 중 현장 수** — 현재 활성 현장/프로젝트 수
- **오늘 할 일** — 금일 예정된 작업 리스트 요약
- **미처리 하자** — 접수되었으나 미해결 하자 건수
- **최근 고객 메시지** — 미확인 고객 문의 수
- **이번 달 매출** — 당월 계약/수금 금액

배치 순서: 현장 현황 > 할 일 > 하자 > 고객 소통 > 매출. "오늘 당장 해야 할 것" 중심 배치.

**차트/시각화:**
- **현장별 진행 상태 리스트** — 카드형으로 각 현장 상태 표시 (진행중/하자보수/완료)
- **월별 매출 바 차트** — 최근 6개월 매출 추이
- **하자 유형별 통계** — 누수, 도배 들뜸, 타일 깨짐 등 유형별 건수
- **일정 캘린더** — 월간/주간 일정 캘린더 뷰

**네비게이션:**
- 하단 탭 바 (모바일) — 홈, 현장, 일지, 채팅, MY
- 비교적 단순한 1-depth 구조
- 현장 선택 후 상세 화면에서 탭으로 세부 기능 접근
- 심플한 아이콘 + 한글 라벨 조합

**모바일 대응:**
- 모바일 전용 앱 (웹 버전 제한적이거나 미제공)
- 시공 사진 before/after 비교 기능
- 하자 접수 시 사진 + 위치 마킹
- 고객에게 시공 진행 상황 공유 링크 발송

**개인화/커스터마이징:**
- 하자 유형 카테고리 커스터마이징
- 시공 일지 양식 일부 커스터마이징
- 대시보드 커스터마이징은 미지원
- 업체 로고/브랜딩 적용 (고객 공유 페이지)

**강점 (인테리어코치 관점):**
- 인테리어 업계 실무에 가장 근접한 하자 관리/시공 일지 기능
- Before/After 사진 비교가 고객 만족도 제고 및 포트폴리오에 활용 가능
- 고객 소통(채팅, 진행 상황 공유)이 내장되어 별도 메신저 불필요
- UI가 비교적 모던하고 직관적

**약점 (인테리어코치 관점):**
- 경영 관리 기능 부재 (견적 관리, 세무, 마케팅)
- 웹 대시보드가 약하거나 미지원 — 사무실에서 전체 현황 파악이 어려움
- 다크 모드 미지원
- 다중 프로젝트 비교/총괄 관리 뷰가 약함
- 매출/재무 분석 기능이 기초적

## 3. 인테리어코치 차별화 포인트 3가지

### 차별화 1: "All-in-One 경영 코치" — 현장부터 세무까지 단일 플랫폼

경쟁사들은 크게 두 진영으로 나뉜다. **해외 SaaS**(Buildertrend, CoConstruct, Houzz Pro)는 프로젝트 관리 + 재무에 강하지만 한국 시장 맥락(세금계산서, 홈택스 연동, 한국형 견적서)이 없고, **국내 앱**(빌드온, 현장의신)은 현장 관리에 특화되지만 견적/마케팅/세무 기능이 없다.

인테리어코치는 **현장 관리 + 견적 코치 + 마케팅(리드) + 세무 연동**을 한국 인테리어 업계 맥락에서 하나로 묶는 유일한 플랫폼이 될 수 있다. 대시보드에서 "오늘 현장 상황 / 이번 달 견적 전환율 / 미수금 / 세금 신고 일정"을 한 화면에 보여주는 것은 어떤 경쟁사도 하지 못하는 것이다.

### 차별화 2: "다크 모드 퍼스트" + 모던 UI — 업계 유일의 프리미엄 UX

분석 대상 5개 서비스 중 다크 모드를 지원하는 서비스는 **단 하나도 없다**. 건설/인테리어 SaaS 시장은 전반적으로 UI가 기능 중심이고 시각적 세련미가 부족하다. 인테리어코치의 **#0A0A0A 다크 배경 + #00C471 그린 액센트** 조합은 업계에서 독보적인 시각적 정체성을 만든다.

현장에서 스마트폰으로 대시보드를 확인할 때 다크 모드는 실용적이기도 하다 (야외 눈부심 감소, 배터리 절약). 이 "프리미엄 SaaS" 느낌은 인테리어 업체 대표들에게 "전문가 도구"로서의 인식을 심어줄 수 있다.

### 차별화 3: "AI 견적 코치" — 데이터 기반 의사결정 지원

Buildertrend이나 Houzz Pro가 데이터를 "보여주는" 데 그친다면, 인테리어코치는 축적된 데이터를 기반으로 **능동적으로 조언하는 코치 역할**을 할 수 있다.

예시:
- "이 견적의 자재 단가가 최근 3개월 평균 대비 15% 높습니다"
- "이번 달 마케팅 전환율이 하락 추세입니다. 포트폴리오 업데이트를 추천합니다"
- "현장 A의 공정이 예정보다 3일 지연 중입니다. 인력 재배치를 검토하세요"

"대시보드가 알려주는 SaaS"에서 **"대시보드가 코칭하는 SaaS"**로의 전환이 인테리어코치만의 핵심 차별점이다.

## 4. 벤치마킹할 UX 패턴 3가지

### 패턴 1: Houzz Pro의 "리드-견적-계약-시공-완료" 파이프라인 뷰

**원본:** Houzz Pro는 대시보드에서 고객 획득(리드)부터 프로젝트 완료까지의 전체 퍼널을 한눈에 보여주는 파이프라인 시각화를 제공한다. 각 단계에 몇 건이 있는지, 전환율은 어떤지를 직관적으로 파악할 수 있다.

**적용 방안:**
인테리어코치 대시보드에 **"비즈니스 파이프라인" 위젯**을 추가한다.

```
문의 유입(5) → 상담 진행(3) → 견적 발송(2) → 계약 완료(1) → 시공 중(4) → 시공 완료(2)
```

각 단계를 수평 바 또는 칸반 카드로 표시하고, 단계별 건수와 금액을 함께 표시한다. 다크 배경에서 각 단계를 #00C471 그라데이션으로 표현하면 시각적으로도 인상적이다. 각 단계 클릭 시 해당 건 목록으로 바로 이동하는 바로가기 역할도 겸한다.

### 패턴 2: Buildertrend의 "오늘의 브리핑" 모닝 대시보드

**원본:** Buildertrend는 로그인 직후 "오늘 해야 할 일"을 최전면에 보여주는 To-Do 중심 대시보드를 제공한다. 빌드온 역시 "오늘 현장 인원/작업"을 가장 먼저 보여준다. 이는 바쁜 현장 대표가 아침에 앱을 열었을 때 즉시 행동할 수 있게 한다.

**적용 방안:**
대시보드 최상단에 **"오늘의 브리핑" 카드**를 배치한다.

```
오늘의 브리핑
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
현장 3곳 진행 중  |  미확인 알림 2건  |  견적 승인 대기 1건
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 09:00  강남 아파트 - 타일 시공 (2일차)
 10:30  판교 빌라 - 자재 입고 확인
 14:00  분당 오피스텔 - 고객 현장 미팅
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

시간대별로 오늘 해야 할 일을 정렬하고, 긴급도에 따라 색상 구분(빨강: 긴급, 노랑: 주의, 녹색: 정상)을 적용한다. 디지털 비숙련 사용자도 "앱 열면 오늘 뭐 해야 하는지 바로 보인다"는 경험을 제공한다.

### 패턴 3: 현장의신의 "Before/After 현장 카드" + CoConstruct의 "선택 사항 진행 바"

**원본:** 현장의신은 시공 사진 Before/After를 나란히 비교하는 UI를 제공하여 시공 진행을 시각적으로 보여준다. CoConstruct는 마감재 선택 사항의 진행 상태를 카테고리별 progress bar로 표시하여 "고객이 아직 몇 가지를 결정하지 않았는지" 한눈에 파악 가능하다.

**적용 방안:**
대시보드의 **"현장 현황" 섹션**에 각 현장 카드를 다음과 같이 구성한다.

```
┌─────────────────────────────────────┐
│  강남 아파트 리모델링                    │
│  ────────────────────────────────── │
│  [Before 썸네일] → [After 썸네일]      │
│  공정률 ████████░░░░ 68%             │
│  자재 선택  ██████████░░ 83% (2건 미결) │
│  D-12 (예정 완공일)                     │
│  ────────────────────────────────── │
│  [일지 작성]  [견적 확인]  [고객 연락]    │
└─────────────────────────────────────┘
```

사진으로 시각적 진행을 보여주고, progress bar로 정량적 진행을 보여주며, 하단 Quick Actions로 바로 행동할 수 있게 한다. 이는 "한 카드 안에서 현장의 모든 것을 파악하고 행동까지 이어지는" 통합 경험을 제공한다.

인테리어코치 대시보드 v2는 해외 SaaS의 **데이터 시각화 성숙도**와 국내 앱의 **현장 실무 밀착형 UX**를 결합하되, **다크 모드 프리미엄 경험**과 **AI 기반 능동적 코칭**으로 어느 쪽에도 없는 독자적 가치를 만들어야 한다.
