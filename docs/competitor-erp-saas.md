## 1. 분석 개요

### 분석 목적

한국 인테리어 업종에서 사용되는 ERP, SaaS, 플랫폼 도구를 6개 카테고리로 분류하여 심층 분석하고, 인테리어코치의 시장 내 최적 포지셔닝 전략을 도출한다.

### 분석 대상 6개 카테고리

| 카테고리 | 대표 서비스 | 성격 |
|---------|-----------|------|
| 아파트 인테리어 정보 플랫폼 | 호갱노노 | B2C 정보 → B2B 확장 |
| 인테리어 중개 플랫폼 | 집닥 | 시공사 중개 + 관리도구 |
| 홈리빙 플랫폼 | 오늘의집 | 커머스 + 시공 중개 |
| 건설현장 특화 ERP | 하우빌드, 건설워커, 현장의신 | 현장 관리 중심 |
| 범용 비즈니스 도구 | 삼쩜삼, 자비스, 얼마에요, Flex | 세무/회계/인사 특화 |
| 해외 시공관리 SaaS | Buildertrend, CoConstruct, Houzz Pro, Jobber | 글로벌 벤치마크 |

### 분석 기준 (9개 핵심 기능)

견적관리, 공정관리, 하자관리, 수금/회계, 고객관리(CRM), 마케팅도구, 근태관리, 모바일앱, AI기능

## 2. 호갱노노 시공관리

| 항목 | 내용 |
|------|------|
| **서비스명** | 호갱노노 (HogangNoNo) |
| **운영사** | 주식회사 호갱노노 |
| **설립** | 2015년 |
| **주요기능** | 아파트 실거래가 조회, 시세 분석, 단지 정보, 인테리어 시공 사례 공유, 시공업체 리뷰 |
| **가격체계** | 기본 무료 (앱 내 광고 수익 + 프리미엄 구독) |
| **타겟고객** | 아파트 매매/전세 관심 소비자 → 인테리어 시공 소비자 |
| **MAU 추정** | 300~500만 |
| **매출 추정** | 연 100~200억 원 |

### 주요 기능 상세

- **아파트 시세/실거래가**: 국토부 데이터 기반 실거래가 조회, 시세 트래킹, 호가 비교
- **단지 정보**: 단지별 세대수, 면적, 주차, 관리비, 커뮤니티 시설 정보
- **시공 관리 기능 (확장)**: 2023년 이후 인테리어 카테고리 강화
  - 아파트 평형별 인테리어 사례 조회
  - 시공업체 리뷰 및 평점 시스템
  - 시공 견적 비교 기능 (일부)
  - 입주 예정 단지별 인테리어 패키지 정보
- **입주 캘린더**: 신축 아파트 입주 일정 → 인테리어 수요 예측 데이터

### 장점
- 압도적인 부동산 관련 트래픽 보유 (MAU 수백만)
- 아파트 단지별 상세 데이터 확보
- 입주 예정 단지 정보 → 인테리어 수요 선행 지표로 활용 가능
- 소비자 신뢰도 높음 (실거래가 데이터 기반)

### 단점
- 시공업체 관리(B2B) 기능이 매우 약함 (소비자 리뷰 수집 수준)
- 견적관리, 공정관리, 하자관리 등 실질적 업무 도구 부재
- 인테리어 업체를 위한 SaaS 도구가 아닌, 소비자 정보 플랫폼 성격
- 빌라, 주택, 상업공간 등 아파트 외 시공 케이스 미지원

### 인테리어코치와의 관계

호갱노노는 **경쟁사라기보다 잠재적 고객 유입 채널**에 가깝다. 입주 예정 단지 데이터를 활용하면, 해당 단지 인근 인테리어 업체에게 인테리어코치 마케팅이 가능하다.

## 3. 집닥 시공사 어드민

| 항목 | 내용 |
|------|------|
| **서비스명** | 집닥 (ZipDock) |
| **운영사** | 주식회사 집닥 |
| **설립** | 2015년 |
| **주요기능** | 인테리어 시공사 중개, 시공사 어드민 패널, 일정관리, 수금관리, 시공 현황 추적 |
| **가격체계** | 시공 중개 수수료 8~15% (성사 기준), 시공사 어드민은 등록 업체 무료 제공 |
| **타겟고객** | 중소 인테리어 시공사 (등록 업체 1,000~3,000개 추정) |
| **MAU 추정** | 소비자 앱 100~200만, 시공사 어드민 수천 명 |
| **매출 추정** | 연 200~400억 원 |

### 소비자 측 (B2C)
- 인테리어 시공사 검색 및 매칭
- 견적 요청 (평형, 스타일, 예산 기반)
- 시공 사례 포트폴리오 조회
- 리뷰 및 평점 시스템
- 시공 보증 서비스 (하자 발생 시 보증금 보상)

### 시공사 측 (B2B - 시공사 어드민)
- **리드 관리**: 소비자 견적 요청 수신, 상담 내역 관리
- **일정 관리**: 시공 스케줄 캘린더, 공종별 일정 배정
- **수금 관리**: 계약금/중도금/잔금 단계별 결제 추적, 미수금 알림
- **시공 현황 보드**: 진행 중/완료/AS 건별 상태 추적
- **견적서 작성**: 템플릿 기반 견적서 생성 및 소비자 공유
- **리뷰 관리**: 시공 완료 후 리뷰 요청 자동 발송
- **정산 대시보드**: 집닥 수수료 정산 내역 확인

### 장점
- 소비자 트래픽 확보를 통한 안정적 리드 공급
- 중개 플랫폼 + 관리도구 통합 (시공사 입장에서 원스톱)
- 시공 보증 서비스로 소비자 신뢰 확보 → 업체 계약 전환율 상승
- 수금 단계별 관리로 업체 현금흐름 가시성 제공

### 단점
- **높은 수수료 부담**: 시공 성사 시 8~15% 수수료는 업체 마진 압박
- **플랫폼 종속**: 자체 마케팅/브랜딩 역량 강화 대신 집닥 의존도 증가
- **제한적 관리 도구**: 공정관리, 하자관리, 자재관리 등 심층 기능 부족
- **세무/회계 연동 없음**: 수금은 추적하나 실제 장부/세무와 미연동
- **마케팅 도구 없음**: 업체 자체 마케팅(블로그, SNS) 지원 전무
- **가격 경쟁 유도**: 소비자에게 복수 견적 제공 → 최저가 경쟁 압박

### 인테리어코치와의 관계

집닥은 인테리어코치의 **간접 경쟁자이자 보완 파트너**이다. 집닥 등록 업체가 인테리어코치의 핵심 타겟 고객이다. 집닥의 높은 수수료와 플랫폼 종속에 불만을 가진 업체에게, 인테리어코치는 "자체 마케팅 + 경영 관리 역량을 갖추어 플랫폼 의존도를 줄이자"는 가치를 제안할 수 있다.

## 4. 오늘의집 시공사 파트너

| 항목 | 내용 |
|------|------|
| **서비스명** | 오늘의집 파트너센터 (전문가 시공 서비스) |
| **운영사** | 주식회사 버킷플레이스 |
| **설립** | 2014년 |
| **주요기능** | 시공사 파트너 대시보드, 리뷰/포트폴리오 관리, 리드 관리, 정산, 시공 사례 콘텐츠 |
| **가격체계** | 시공 중개 수수료 10~15%, 커머스 광고비 별도 |
| **타겟고객** | 인테리어 시공사, 인테리어 디자이너, 부분시공 전문가 |
| **MAU 추정** | 소비자 앱 800~1,200만, 파트너센터 수만 업체 |
| **매출 추정** | 연 3,000~5,000억 원 (커머스+시공+광고 합산) |

### 시공사 파트너 측 (B2B)
- **파트너 대시보드**: 견적 요청 현황, 계약 현황, 매출 통계
- **리뷰 관리**: 고객 리뷰 확인/답변, 리뷰 점수 관리
- **포트폴리오 관리**: 시공 사례 사진/영상 업로드, 스타일별 카테고리
- **리드 관리**: 견적 요청 수신, 상담 이력 관리, 계약 전환 추적
- **정산 관리**: 시공 완료 건별 수수료 정산 내역
- **프로모션**: 플랫폼 내 상위 노출 광고 (CPC/CPM 기반)

### 장점
- **압도적 트래픽**: 국내 최대 홈리빙 플랫폼 (MAU 1,000만 이상)
- **콘텐츠 생태계**: 시공 사례가 마케팅 콘텐츠 역할, 자연 유입 발생
- **브랜드 신뢰도**: "오늘의집에서 찾은 업체"라는 소비자 신뢰
- **커머스 시너지**: 자재 구매까지 연결되는 풀 퍼널

### 단점
- **높은 수수료**: 10~15% 시공 수수료로 업체 마진 압박
- **플랫폼 종속 심화**: 오늘의집 리뷰/사례가 자산이지만 플랫폼 이탈 시 소멸
- **가격 하향 압력**: 소비자에게 복수 견적 비교 → 최저가 경쟁
- **실질적 경영 도구 부재**: 공정관리, 하자관리, 원가 분석, 세무/회계 기능 없음
- **CRM 기능 부재**: 고객과의 지속적 관계 관리 도구 없음

### 인테리어코치와의 관계

오늘의집은 인테리어코치의 **가장 주요한 간접 경쟁자이자 잠재적 위협**이다. 오늘의집이 파트너센터를 본격적인 B2B SaaS로 확장할 경우 직접 경쟁이 발생한다. 그러나 현재까지는 "중개 플랫폼"에 집중하고 있어, "오늘의집에 등록하되, 자체 역량도 키우자"는 보완적 포지셔닝이 가능하다.

## 10. 전략적 시사점

### 경쟁 환경 핵심 발견

1. **파편화된 시장**: 한국 인테리어 업체는 현장관리/마케팅/세무/CRM을 각각 다른 도구로 사용 중이며, 이를 통합한 서비스가 없다.
2. **플랫폼 종속 피로감**: 집닥/오늘의집의 높은 수수료(8~15%)와 가격 경쟁 유도에 대한 불만이 크다.
3. **해외에는 이미 존재**: Buildertrend, Houzz Pro 등 해외에서는 이미 올인원 시공관리 SaaS가 성공적이다. 한국에는 아직 동급 서비스가 없다.
4. **AI 전환 시점**: 기존 서비스 대부분은 AI 기능이 없거나 기초적 수준이다.
5. **가격 경쟁력**: 인테리어코치 Pro(23.9만 원)는 개별 도구 합산 비용의 1/5~1/15 수준이다.

### 잠재적 위협 시나리오

| 위협 시나리오 | 발생 확률 | 대응 전략 |
|-------------|----------|----------|
| 오늘의집이 파트너센터를 B2B SaaS로 확장 | 중간 (2~3년) | 선점 + 데이터 모트 구축 + 자체 마케팅 차별화 |
| 하우빌드가 마케팅 기능 추가 | 낮음 | 인테리어 특화 깊이로 차별화 |

### 인테리어코치 로드맵 시사점

**즉시 차별화 가능 영역** (현재 경쟁사에 없는 것)
- AI 마케팅 콘텐츠 자동 생성 (블로그, SNS, 유튜브)
- AI 견적 분석 (B2C 유입 엔진)
- 인테리어 업종 특화 세무 가이드
- 현장별 수익성 자동 분석

**중기 강화 영역** (해외 벤치마크 수준 도달)
- 고객 포털 (Buildertrend 벤치마크) - 고객이 직접 공정 확인
