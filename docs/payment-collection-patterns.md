## 1. 인테리어 업종 결제 조건

### 1-1. 표준 결제 단계

#### 소형 공사 (1,000만 원 이하) - 2단계

| 단계 | 비율 | 시점 | 비고 |
|------|------|------|------|
| 계약금 | 50% | 계약 시 | 자재 선구매 비용 충당 |
| 잔금 | 50% | 완공 후 | 검수 완료 후 지급 |

#### 중형 공사 (1,000~5,000만 원) - 3단계

| 단계 | 비율 | 시점 | 비고 |
|------|------|------|------|
| 계약금 | 30~40% | 계약 시 | 자재비 + 착수 비용 |
| 중도금 | 30~40% | 공정 50% 도달 | 철거+골조 완료 시점 |
| 잔금 | 20~30% | 완공 후 | 검수 완료 후 7일 내 |

**가장 흔한 패턴: 40-30-30** (계약금 40%, 중도금 30%, 잔금 30%)

#### 대형 공사 (5,000만 원 이상) - 4단계

| 단계 | 비율 | 시점 | 비고 |
|------|------|------|------|
| 계약금 | 30% | 계약 시 | |
| 1차 중도금 | 30% | 철거/기초 완료 | |
| 2차 중도금 | 20% | 마감 착수 시 | |
| 잔금 | 20% | 완공 검수 후 | 하자보수보증금 5% 별도 유보 가능 |

### 1-2. 고객 유형별 차이

| 구분 | 개인 (B2C) | 법인 (B2B) | 건설사 하도급 |
|------|-----------|-----------|-------------|
| 결제 단계 | 2~3단계 | 3~4단계 | 기성금 방식 |
| 계약금 비율 | 30~50% | 20~30% | 10~20% |
| 잔금 비율 | 20~30% | 20~30% | 20~30% |
| 결제 수단 | 계좌이체 위주 | 계좌이체+세금계산서 | 어음+세금계산서 |
| 결제 기한 | 즉시~7일 | 30~60일 | 60~90일 |
| 미수 리스크 | 낮음 | 보통 | 높음 |

### 1-3. 흔한 결제 분쟁 패턴

| 분쟁 유형 | 빈도 | 원인 | 예방법 |
|----------|------|------|--------|
| 잔금 미지급 | 매우 흔함 | 하자 핑계, 추가공사 불만 | 검수 체크리스트 활용, 서면 합의 |
| 추가 비용 분쟁 | 흔함 | 구두 합의 후 부인 | 변경주문서 서면 처리 |
| 중도금 지연 | 보통 | 자금 사정, 공정 불만 | 공정 사진 공유로 신뢰 구축 |
| 계약 취소 | 드묾 | 마음 변경, 이사 취소 | 계약서에 위약금 조항 명시 |
| 하자보수 후 감액 요구 | 보통 | 하자 과대 주장 | 하자 접수 체계화, 사진 기록 |

## 2. 미수금 관리

### 2-1. 업종 평균 통계 (추정)

| 지표 | 개인 고객 (B2C) | 법인 고객 (B2B) | 하도급 |
|------|----------------|----------------|--------|
| 평균 수금 기간 | 완공 후 7~14일 | 세금계산서 발행 후 30~45일 | 60~90일 |
| 미수금 발생률 | 10~15% | 15~25% | 20~30% |
| 대손 (회수 불가) 비율 | 1~3% | 3~5% | 5~8% |
| 평균 미수 기간 | 30일 | 60일 | 90일 |

### 2-2. 미수금 발생 주요 원인

| 원인 | 비율 | 설명 |
|------|------|------|
| 하자/품질 불만 | 35% | "하자 안 고쳐주면 돈 안 내겠다" |
| 자금 사정 | 25% | 고객의 일시적 자금 부족 |
| 추가 비용 분쟁 | 20% | 예상보다 비용이 커서 일부 미지급 |
| 의도적 미지급 | 10% | 악의적 미지급 |
| 연락 두절 | 10% | 시공 완료 후 연락 차단 |

### 2-3. 법적 수금 절차

| 단계 | 방법 | 비용 | 소요 기간 | 설명 |
|------|------|------|----------|------|
| 1단계 | 우호적 독촉 | 0원 | 1~2주 | 전화, 카카오톡, 문자 |
| 2단계 | 내용증명 | 5,000~10,000원 | 1주 | 우체국 내용증명 발송 |
| 3단계 | 지급명령 | 인지대+송달료 (~5만 원) | 2~4주 | 법원 지급명령 신청 |
| 4단계 | 소액사건심판 | 인지대 (~2~3만 원) | 1~2개월 | 3,000만 원 이하 소송 |
| 5단계 | 민사소송 | 인지대+변호사 비용 | 3~12개월 | 3,000만 원 초과 |
| 6단계 | 강제집행 | 별도 비용 | 1~3개월 | 판결 후 재산 압류 |

### 2-4. 미수금 예방 체크리스트

- [ ] 계약서에 결제 일정/조건 명확히 기재
- [ ] 각 결제 단계 전 검수 확인서 받기
- [ ] 추가/변경 공사는 반드시 서면 합의
- [ ] 공정 사진을 실시간 공유 (신뢰 구축)
- [ ] 잔금 지급 전까지 열쇠/비밀번호 미전달
- [ ] 하자 접수 채널을 명확히 안내
- [ ] 계약금 비율을 30% 이상으로 설정
- [ ] 단골 고객에게도 결제 조건 동일 적용

## 3. 세금계산서 타이밍

### 3-1. 부가가치세법 상 공급시기

| 거래 유형 | 공급시기 (세금계산서 발급 시점) | 근거 |
|----------|---------------------------|------|
| 재화 공급 | 재화가 인도되는 때 | 부가세법 제15조 |
| 용역 공급 | 역무 제공이 완료되는 때 | 부가세법 제16조 |
| 중간지급 | 대가의 각 부분을 받기로 한 때 | 부가세법 제17조 |
| 완성도 기준 | 작업 진행 정도에 따라 | 부가세법 제17조 |

**인테리어 시공 적용:**
- **계약금 수령 시**: 선발행 가능 (부가세법 제17조 "대가의 각 부분을 받기로 한 때")
- **중도금 수령 시**: 해당 금액에 대해 발급
- **잔금 수령 시 (완공)**: 최종 세금계산서 발급

### 3-2. 선발행 vs 후발행

| 구분 | 선발행 | 후발행 |
|------|--------|--------|
| **시점** | 대금 수령 전/수령 시 | 시공 완료 후 |
| **장점** | 현금 흐름 관리 용이, 매입세액 공제 빠름 | 정확한 금액 기재 가능 |
| **단점** | 추가 비용 발생 시 수정 필요 | 매입세액 공제 지연 |
| **적합 케이스** | 계약금/중도금 수령 시 | 잔금 (최종 정산) |
| **가산세 리스크** | 과다 발행 시 수정 필요 | 미발행/지연 시 가산세 |

### 3-3. 전자세금계산서 의무발급

| 구분 | 기준 | 의무 |
|------|------|------|
| 법인사업자 | 모든 법인 | 전자세금계산서 의무 |
| 개인사업자 | 전년도 총 수입 **8,000만 원 이상** | 전자세금계산서 의무 |
| 개인사업자 | 전년도 총 수입 8,000만 원 미만 | 종이 세금계산서 가능 |

**미발급 가산세**: 공급가액의 **2%**
**지연발급 가산세**: 공급가액의 **1%** (공급시기 이후 확정신고기한 내)

### 3-4. 간이과세자 vs 일반과세자

| 구분 | 일반과세자 | 간이과세자 |
|------|-----------|-----------|
| **기준** | 연매출 8,000만 원 이상 | 연매출 8,000만 원 미만 |
| **세금계산서 발급** | 의무 (전자/종이) | 발급 불가 (영수증 발급) |
| **부가세율** | 10% | 업종별 1.5~4% |
| **매입세액 공제** | 전액 공제 | 제한적 공제 |
| **인테리어 업체 현실** | 대부분 일반과세자 | 1인 소규모 시공만 해당 |

## 4. 결제 수단 비율

### 4-1. 인테리어 업종 결제 수단 비중 (추정)

| 결제 수단 | 비율 | 수수료 | 비고 |
|----------|------|--------|------|
| 계좌이체 | 55~65% | 없음 | 가장 보편적 |
| 카드결제 | 15~25% | 1.5~3.5% | 소비자 요구 증가 추세 |
| 현금 | 5~10% | 없음 | 감소 추세 (증빙 이슈) |
| 에스크로 | 3~5% | 플랫폼별 상이 | 집닥/오늘의집 중개 시 |
| 인테리어 대출 | 3~5% | 고객 부담 | 카카오뱅크, 하나캐피탈 등 |
| 어음/전자결제 | 1~3% | 별도 | 하도급/법인 거래 |

### 4-2. 카드결제 수수료 구조

| 구분 | 수수료율 | 비고 |
|------|---------|------|
| 영세 사업자 (3억 이하) | 0.5~1.5% | 우대 수수료 |
| 중소 사업자 (3~10억) | 1.5~2.5% | 일반 수수료 |
| 일반 사업자 (10억 초과) | 2.0~3.5% | 업종별 차등 |

**인테리어 업체 카드결제 현실:**
- 고액 거래(1,000만 원+)를 카드로 결제하면 수수료 15~35만 원
- 많은 업체가 카드결제를 기피하지만, 소비자기본법상 거절 불가
- 카드결제 요청 시 "부가세 별도" 또는 "카드 수수료 추가" 요청하는 관행 (불법)

### 4-3. 에스크로/안전결제

| 플랫폼 | 방식 | 수수료 | 고객 보호 |
|--------|------|--------|----------|
| 집닥 | 시공 보증금 + 단계별 지급 | 8~15% (시공비에 포함) | 하자 시 보증금 보상 |
| 오늘의집 | 플랫폼 결제 → 시공사 정산 | 10~15% | 불만 시 환불 중재 |
| 숨고 | 결제 보호 | 플랫폼 수수료 포함 | 서비스 미이행 시 환불 |
| 네이버 페이 | 에스크로 결제 | PG 수수료 | 미배송/미이행 보호 |

### 4-4. 인테리어 대출 상품

| 상품 | 금리 | 한도 | 특징 |
|------|------|------|------|
| 카카오뱅크 인테리어 대출 | 연 4~8% | 최대 5,000만 원 | 비대면, 빠른 승인 |
| 하나캐피탈 인테리어론 | 연 5~10% | 최대 3,000만 원 | 시공사 제휴 할인 |
| NH농협 인테리어 대출 | 연 4~7% | 최대 5,000만 원 | 아파트 담보 연계 |
| 카드사 할부 | 월 0.6~0.8% | 카드 한도 내 | 장기 할부 시 수수료 높음 |

## 5. 수금관리 기능 제안

### 5-1. MVP 기능 목록

| 기능 | 우선순위 | 설명 |
|------|---------|------|
| 결제 일정 관리 | P0 | 프로젝트별 계약금/중도금/잔금 일정 등록 |
| 수금 상태 대시보드 | P0 | 수금 예정/완료/미수 현황 한눈에 보기 |
| 수동 수금 확인 | P0 | 입금 확인 시 수금 완료 처리 |
| 미수금 알림 | P0 | 결제 예정일 전/후 자동 알림 (사장님에게) |
| 결제 이력 조회 | P0 | 프로젝트별/고객별/기간별 결제 이력 |
| 고객 알림 (카카오/문자) | P1 | 결제 안내/독촉 알림 고객 발송 |
| 월별 수금 현황 리포트 | P1 | 월별 예정/실수금/미수 요약 |
| 세금계산서 발급 연동 | P1 | 수금 확인 시 세금계산서 자동 발급 |
| 견적서에서 청구서 변환 | P1 | 견적서 기반 청구서 자동 생성 |
| 미수 에이징 분석 | P2 | 미수금 경과일수별 분류 (30/60/90일) |

### 5-2. 통합 프로세스 흐름

```
[견적서 작성]
     | (고객 승인)
[계약서 생성] ← 견적서 정보 자동 반영
     | (결제 일정 자동 생성)
[결제 일정 등록]
     |-- 계약금: 금액, 예정일
     |-- 중도금: 금액, 예정일, 공정 조건
     +-- 잔금: 금액, 예정일, 검수 조건
          |
[청구서 발행] ← 결제 일정 기반 자동 생성
     | (고객에게 발송: 카카오/문자/이메일)
[수금 대기]
     |-- D-7: 사전 안내 알림 (자동)
     |-- D-day: 결제일 리마인더 (자동)
     +-- D+3: 미입금 알림 (자동)
          |
[수금 확인] ← 수동 확인 또는 가상계좌 자동 매칭
     |
[세금계산서 발급] ← 수금 확인 후 자동 발급
     |
[완료] → 수금 이력 기록, 리포트 반영
```

### 5-3. 알림 자동화 규칙

| 시점 | 대상 | 채널 | 내용 |
|------|------|------|------|
| D-7 | 사장님 | 인앱 | "[프로젝트명] 계약금 수금 예정 (7일 후)" |
| D-3 | 고객 | 카카오 | "[업체명] 인테리어 공사 결제 안내" |
| D-day | 사장님 + 고객 | 인앱+카카오 | "오늘 결제 예정일입니다" |
| D+3 | 사장님 | 인앱 | "미수금 발생: [프로젝트명] 3일 경과" |
| D+7 | 고객 | 카카오/문자 | "결제 확인 요청" (우호적) |
| D+14 | 사장님 | 인앱 | "미수금 경고: [프로젝트명] 14일 경과" |
| D+30 | 사장님 | 인앱 | "미수금 심각: 법적 조치 검토 안내" |

### 5-5. 외부 서비스 연동 계획

| 서비스 | 용도 | 우선순위 | 비용 |
|--------|------|---------|------|
| 팝빌 | 전자세금계산서 발급 | P1 | 건당 100~200원 |

**Phase 1 (MVP) - 4~6주**
- 결제 일정 CRUD (계약금/중도금/잔금 등록)
