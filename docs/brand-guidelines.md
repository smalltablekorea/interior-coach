# 인테리어코치 브랜딩 가이드라인

> Brand Identity & Design System | v1.0 | 2026.03.23

---

## 1. 브랜드 에센스

### 미션
인테리어 업체의 모든 업무를 하나로 통합하여, 시공에만 집중할 수 있게 한다.

### 비전
대한민국 인테리어 업체의 디지털 전환을 이끄는 올인원 플랫폼.

### 핵심 가치
- **통합 (All-in-One)**: 4가지 업무를 하나로
- **실용 (Practical)**: 현장 경험에서 나온 실제 기능
- **신뢰 (Trustworthy)**: 실제 시공 데이터 기반
- **접근성 (Accessible)**: 디지털에 익숙하지 않아도 쉽게

### 브랜드 슬로건
> **"인테리어 업체의 모든 업무, 하나로."**

서브 슬로건:
- "현장관리부터 마케팅까지, 하나로."
- "블로그 대행 70만원 → 0원"
- "인테리어 업체를 위한 올인원 SaaS"

## 2. 브랜드 보이스

### 톤 & 매너

**핵심 원칙: 전문적이면서 친근한 동료의 톤**

인테리어 업체 대표에게 말하는 **경험 있는 동료** 느낌. 위에서 가르치는 게 아니라, 옆에서 같이 일하는 사람.

| 성격 | 이런 느낌 O | 이런 느낌 X |
|------|-----------|-----------|
| **전문적** | "공종별 정산이 실시간으로 됩니다" | "최첨단 AI 기반 다차원 분석 엔진" |
| **친근한** | "대표님, 이번 달 부가세 신고 준비되셨나요?" | "귀하의 세무 일정을 안내드립니다" |
| **실용적** | "블로그 대행 비용 0원으로 만드는 법" | "혁신적 마케팅 자동화 솔루션" |
| **솔직한** | "현실적으로 월 100만 원은 아끼실 수 있어요" | "무한한 비용 절감 가능성" |

### 말투 가이드

#### SNS / 마케팅
- **존댓말 기본** (해요체)
- 과한 이모지 자제, 핵심에 1~2개
- 숫자는 구체적으로 (월 95만 원 → 29.9만 원)
- 업종 용어 자연스럽게 사용 (공종, 실행가, 원가율)

```
Good: "30평 아파트 인테리어, 평균 견적 2,500만 원인데요. 이 금액이 적정한지 3분 만에 확인할 수 있어요."
Bad: "혁신적 AI 기술로 인테리어 견적을 분석해보세요! 🚀🔥💪"
```

#### 제품 내 (UI 텍스트)
- **간결하고 명확**하게
- 동사형 CTA ("분석하기", "시작하기", "저장하기")
- 에러 메시지도 친근하게 ("다시 시도해주세요" → "잠깐 문제가 생겼어요. 한 번 더 시도해주시겠어요?")

#### 고객 지원 / CS
- **공감 우선** → 해결 방안 제시
- "불편을 드려 죄송합니다" + 구체적 해결 방법

## 3. 컬러 시스템

### Primary Color — Green

인테리어코치의 메인 컬러. 성장, 신뢰, 전문성을 상징.

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--green` (Primary) | `#00C471` | CTA 버튼, 강조, 링크, 성공 상태 |
| `--green-light` | `#00C471/10` | 배경 하이라이트, 카드 배경 |
| `--green-dark` | `#00A35E` | 호버 상태, 진한 강조 |

### Secondary Color — Blue

스타터 플랜, 보조 UI 요소.

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--blue` | `#3B82F6` (blue-500) | 스타터 플랜 강조, 링크 보조 |
| `--blue-light` | `#3B82F6/10` | 스타터 관련 배경 |

### Accent Color — Amber

엔터프라이즈, 프리미엄 느낌.

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--amber` | `#F59E0B` (amber-500) | 엔터프라이즈 강조, 배지, PREMIUM 태그 |

### Neutral — Dark Theme Base

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--background` | `#0A0A0A` | 페이지 배경 |
| `--card` | `#141414` | 카드 배경 |
| `--border` | `#ffffff/10` | 구분선, 테두리 |
| `--muted` | `#ffffff/50` | 보조 텍스트 |
| `--foreground` | `#FFFFFF` | 기본 텍스트 |

### Semantic Colors

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--red` | `#EF4444` | 에러, 경고, 비용 강조 |
| `--yellow` | `#EAB308` | 주의, 대기 상태 |
| `--green` | `#00C471` | 성공, 완료, 절감 |

### 컬러 사용 비율
- **Green (Primary)**: 20% — CTA, 강조, 핵심 요소
- **Dark Neutral**: 60% — 배경, 카드, 구조
- **White/Light**: 15% — 텍스트, 아이콘
- **Accent (Blue/Amber)**: 5% — 플랜 구분, 특수 요소

## 5. 로고 사용 가이드

### 로고 구성

```
[■] 인테리어코치
 IC

■ = 녹색(#00C471) 라운드 사각형 (rounded-lg)
IC = 흰색 Bold 텍스트
```

### 로고 버전

| 버전 | 사용처 |
|------|--------|
| 풀 로고 (아이콘 + 텍스트) | 웹 헤더, 랜딩 페이지, 문서 |
| 아이콘만 (IC 뱃지) | 파비콘, 앱 아이콘, 축소 사이드바 |
| 텍스트만 | 본문 내 언급, 제한된 공간 |

### 여백 규칙

- 최소 여백: 아이콘 높이의 **50%**를 사방에 확보
- 로고 위에 다른 요소가 겹치지 않도록

### 최소 크기

| 매체 | 최소 크기 |
|------|----------|
| 디지털 (풀 로고) | 높이 24px 이상 |
| 디지털 (아이콘) | 16×16px 이상 |
| 인쇄 (풀 로고) | 높이 10mm 이상 |

### 배경별 사용

| 배경 | 로고 버전 |
|------|----------|
| 어두운 배경 (#0A0A0A 등) | 기본 (녹색 아이콘 + 흰색 텍스트) |
| 밝은 배경 (#FFFFFF 등) | 녹색 아이콘 + 검정 텍스트 |
| 녹색 배경 | 흰색 아이콘 + 흰색 텍스트 |
| 사진/이미지 위 | 반투명 배경 박스 + 기본 로고 |

### 금지 사항
- 로고 색상 임의 변경
- 로고 비율 변경 (가로/세로 비율 유지)
- 로고 위에 텍스트/이미지 겹침
- 저해상도 사용 (2x 이상 권장)

| 차트/그래프 | 미니멀, 녹색 계열 |
| 카드/박스 | 라운드 코너(16px), 얇은 보더 |

### 콘텐츠 템플릿

#### Threads 포스팅 구조
```
[후킹 첫 줄] — 질문 또는 충격적 사실
[본문 2~3줄] — 인사이트 또는 데이터
[마무리 1줄] — CTA 또는 질문 유도
```

예시:
```
인테리어 업체 10곳 중 9곳이 아직 엑셀로 견적을 씁니다.

블로그 대행에 매달 70만원, 세무사에 35만원, 현장관리앱에 15만원.
이걸 전부 합치면 월 120만원인데, 하나로 통합하면 29만원.

혹시 지금 따로따로 쓰고 계신 건 아닌가요?
```

#### Instagram 피드 구조
```
슬라이드 1: 후킹 이미지 + 텍스트
슬라이드 2~4: 핵심 정보 (숫자, 비교, 방법)
슬라이드 5: CTA ("프로필 링크에서 무료 시작")
```

## 7. 브랜드 아키텍처

```
스몰테이블 (운영사)
├── 인테리어코치 (SaaS 플랫폼)
│   ├── 견적코치 (B2C 견적 분석 서비스)
│   └── 인테리어코치 Pro/Enterprise (B2B 업체 SaaS)
├── Small Table Design (하이엔드 인테리어)
├── Green Avenue (실속형 인테리어)
└── 이매동신 (상업공간 인테리어)
```

- **인테리어코치**가 메인 브랜드, **견적코치**는 서브 브랜드
- 시공 브랜드(Small Table 등)는 별도 운영하되, 인테리어코치의 실제 사용 사례로 활용
- 스몰테이블은 법인명으로만 노출 (Footer, 사업자 정보)
