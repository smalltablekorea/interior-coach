## 키워드 전략 매트릭스

### 검색량 × 경쟁도 우선순위

```
검색량 높음
    │
    │  ★ 아파트 인테리어          ★ 인테리어 비용
    │    (경쟁 높음, 장기)          (경쟁 높음, 장기)
    │
    │  ★ 인테리어 견적 비교        ★ 인테리어 견적
    │    (경쟁 중, 우선)             (경쟁 중, 우선)
    │
    ├──────────────────────────────────────────
    │
    │  ★ 인테리어 시뮬레이션       ★ 현장관리 앱
    │    (경쟁 낮음, 즉시!)          (경쟁 낮음, 즉시!)
    │
    │  ★ 인테리어 마케팅           ★ 인테리어 견적 프로그램
    │    (경쟁 낮음, 즉시!)          (경쟁 낮음, 즉시!)
    │
검색량 낮음                                    경쟁 낮음
```

### 공략 순서

**Phase 1 (즉시)** — 경쟁 낮음 + 전환율 높음
- 인테리어 업체 관리, 현장관리 앱, 인테리어 시뮬레이션
- 인테리어 마케팅, 인테리어 블로그 대행, 인테리어 견적 프로그램
- 인테리어 세무, 인테리어 정산, 인테리어 원가 관리

**Phase 2 (1~2개월)** — 경쟁 중간 + 검색량 중간
- 인테리어 견적, 인테리어 견적 비교, 인테리어 견적 적정가
- 30평/20평 아파트 인테리어 비용, 인테리어 견적서 양식
- 인테리어 바가지, 인테리어 업체 선택 기준

**Phase 3 (3~6개월)** — 경쟁 높음 + 검색량 높음 (도메인 권위 구축 후)
- 인테리어 비용, 아파트 인테리어, 인테리어 업체 추천
- 신축 아파트 인테리어, 아파트 리모델링 비용

## 수치 근거

- 월간 검색량은 네이버 키워드 도구(searchad.naver.com) 기반 추정치
- 실제 검색량은 계절 변동 있음 (이사 성수기 3~5월, 9~10월 증가)
- 경쟁도는 네이버 블로그 상위 노출 난이도 기준 (해당 키워드 블로그 발행량, 대형 사이트 점유율 등)
- 최신 정확 수치는 네이버 검색광고 키워드 도구에서 직접 조회 필요
