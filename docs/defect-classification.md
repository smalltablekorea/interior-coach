# 인테리어 하자 유형 분류 체계

> 인테리어코치 하자관리 기능을 위한 분류 체계 벤치마크 | 2026.03.29 작성

---

## 1. 개요

### 목적
인테리어 시공 후 발생하는 하자를 체계적으로 분류하여, 인테리어코치 앱의 하자 접수/추적/보수 기능에 사용할 드롭다운 데이터를 구축한다.

### 분류 기준
- **공종(Trade)**: 10개 대분류 (도배, 바닥재, 타일, 목공, 전기, 설비, 도장, 창호, 방수, 기타)
- **하자 유형(Defect Type)**: 공종별 5~10개 세부 유형
- **심각도(Severity)**: 1~5 등급
- **발생 빈도(Frequency)**: 상/중/하
- **처리 기간**: 평균 보수 소요일

---

## 2. 관련 법적 기준

### 주택법 / 공동주택관리법

| 공종 | 하자보수 책임기간 | 근거 |
|------|-----------------|------|
| 마감공사 (도배, 도장, 타일 등) | **2년** | 공동주택관리법 시행령 별표 4 |
| 창호공사 | **2년** | 동법 |
| 목공사 | **2년** | 동법 |
| 방수공사 | **3년** (지하외벽 5년) | 동법 |
| 급배수/위생설비 | **2년** | 동법 |
| 전기/통신설비 | **2년** | 동법 |
| 난방설비 (보일러 등) | **2~3년** | 동법 |
| 구조체 (기둥, 벽체, 슬래브) | **5~10년** | 동법 |

### 민법 하자담보책임
- **도급계약 하자담보책임**: 민법 제667~671조
- 건물/공작물 하자: **인도 후 5년** 내 권리 행사
- 토지/건물 외 공작물: **인도 후 1년** 내
- **소멸시효**: 하자를 안 날로부터 **6개월** 내 행사

### 소비자분쟁해결기준 (공정거래위원회)
- 인테리어 시공 하자 → 시공업체 무상 보수 의무
- 보수 불가 시 → 손해배상
- 중대 하자 → 계약 해제 가능

## 6. 앱 하자관리 기능 설계 가이드

### 6-1. 하자 접수 플로우

```
[고객/현장관리자]
    ↓
1. 공종 선택 (드롭다운: 10개)
    ↓
2. 하자 유형 선택 (드롭다운: 공종별 5~10개)
    ↓
3. 위치 표시 (도면 마크업 or 사진 마크업)
    ↓
4. 사진 촬영 (필수, 최소 1장)
    ↓
5. 심각도 자동 제안 (유형별 기본값) + 수동 조정
    ↓
6. 설명 입력 (선택사항, 음성 입력 가능)
    ↓
7. 접수 완료 → 담당자 알림
```

### 6-2. 하자 상태 흐름

```
접수(reported) → 확인(confirmed) → 보수중(in_repair) → 검수대기(pending_review) → 완료(resolved)
                                                                                      ↓
                                                                              재시공(reopened)
```

### 6-3. 사진 촬영 가이드

| 촬영 항목 | 설명 |
