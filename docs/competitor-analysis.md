## 2. 견적 분석 영역: 집첵 (ZipCheck)

### 서비스 개요
인테리어 견적서를 AI + 전문가가 분석하여 적정가 여부를 판단하는 **소비자(B2C)** 서비스.

### 기능 비교

| 기능 | 집첵 | 인테리어코치 (견적코치) |
|------|------|----------------------|
| AI 견적 분석 | O (기본 무료/저가) | O (무료 분석 제공) |
| 공종별 적정가 비교 | O | O (12개 공종 상세) |
| 누락 항목 체크 | O | O |
| 전문가 1:1 상담 | O (유료, 3~10만 원) | O (프로 분석 39,900원) |
| 업체 연결/추천 | O (일부) | **O (SaaS 가입 업체 자동 매칭)** |
| 업체용 SaaS 기능 | X | **O (올인원 업체 관리)** |

### 집첵의 강점
- 시장 검증 완료: 소비자 인테리어 견적 분석 니즈 입증
- 앱스토어 평점 약 4.5~4.7점, 리뷰 수백~수천 건
- 소비자 사이 높은 인지도

### 집첵의 약점
- **B2C Only**: 업체(B2B) 수익 모델 부재
- 견적 분석 후 업체 연결이 핵심 수익인데, 양질의 업체 풀 확보가 과제
- 복잡한 맞춤 시공 견적은 AI만으로 한계
- 분석 서비스만으로는 반복 수익(Recurring Revenue) 구조 취약

### 인테리어코치의 차별화
- 견적코치(B2C)는 **고객 유입 엔진**, 인테리어코치(B2B)가 **수익 엔진**
- 실제 시공 원가 데이터 기반 (자체 3개 브랜드 운영)
- 견적 분석 → 업체 매칭 → 업체 SaaS 구독의 **풀 퍼널** 구현

### 오늘의집

| 항목 | 내용 |
|------|------|
| 포지션 | 국내 최대 인테리어/홈리빙 플랫폼 (MAU ~1,000만) |
| 시공 파트너 | 수천~1만+ 업체 등록 |

| 업체에 주는 가치 | 대규모 트래픽, 리뷰 기반 신뢰 구축 |
| 업체의 페인포인트 | 높은 수수료, 가격 경쟁, 플랫폼 의존도, 자체 브랜딩 약화 |

### 숨고

| 항목 | 내용 |
|------|------|
| 포지션 | 생활 서비스 전문가 매칭 (인테리어 전문가 수만 명) |
| 과금 | 견적 열람 시 코인 차감 (건당 3,000~15,000원) |
| 업체에 주는 가치 | 저비용 리드 획득, 리뷰 시스템 |
| 업체의 페인포인트 | 극심한 가격 경쟁, 계약 불발 시 코인 소진, 고객 품질 편차 |

### 인테리어코치와의 관계
- 오늘의집/숨고는 **경쟁자가 아닌 보완 관계**
- 이들 플랫폼에 등록한 업체가 인테리어코치의 **타겟 고객**
- 인테리어코치를 통해 자체 마케팅 역량을 갖추면 플랫폼 의존도를 줄일 수 있음
- 장기적으로 견적코치 → 업체 매칭 시 **자체 플랫폼** 역할

## 8. 핵심 인사이트

### 경쟁 우위 (Moat)

1. **데이터 모트**: 자체 3개 인테리어 브랜드 운영으로 축적된 실제 원가 DB
2. **네트워크 효과**: 견적코치 소비자 ↔ SaaS 업체 양면 네트워크
3. **올인원 통합**: 4개 영역을 하나로 묶은 국내 유일 서비스
4. **전환 비용**: 현장/고객/견적/세무 데이터가 쌓일수록 이탈 어려움

### 위험 요소

1. **오늘의집의 B2B SaaS 진출 가능성**: 트래픽 + 자본력으로 유사 서비스 런칭 가능
2. **각 영역 전문 서비스의 기능 확장**: 시공왕이 마케팅을 추가하거나, 삼쩜삼이 업종 특화하는 경우
3. **가격 민감도**: 영세 업체의 월 23.9만 원 부담 (→ Free/Starter로 단계적 진입 유도)

### 결론
현재 국내 인테리어 업체 SaaS 시장은 **각 영역별 파편화** 상태이며, 올인원 통합 솔루션은 인테리어코치가 유일하다. 각 개별 서비스 대비 기능은 동등 이상이면서 총비용은 1/3~1/5 수준으로, **강력한 가격 경쟁력**을 확보하고 있다.
