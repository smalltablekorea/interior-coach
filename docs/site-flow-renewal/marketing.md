## 2. 인앱 신기능 안내문

### 2-1. 상단 배너 (한 줄)

```
🆕 현장 한 번 등록하면 공정·일정·계약·납부일정이 자동 — 같은 내용 다시 안 적어도 됩니다.
```

### 2-2. 첫 진입 모달

**헤드라인**
```
현장 등록, 이제 한 번이면 됩니다
```

**서브헤드**
```
현장명·평수·시작일만 넣으시면 공정 14개, 일정 캘린더,
계약·납부일정 4단계가 자동으로 깔립니다.
```

**본문 (3줄, 글머리표)**
- 공정 순서·일수는 32평 기준으로 깔리고, 사장님이 +/- 조정
- 대금은 계약금/착수금/중도금/잔금 기본 4단계 — 2~6개까지 자유 분할
- 자재·작업자·고객 정보가 같은 화면에 묶입니다. 따로 옮겨 적기 안 합니다

**CTA Primary**: `현장 한 번에 등록 시작`
**CTA Secondary**: `나중에 보기`

## 3. 인테리어코치 기능 안내 페이지 (랜딩/내부 안내)

### 3-1. 히어로

**Eyebrow**
```
인테리어코치 업데이트
```

**헤드라인 (2줄)**
```
현장 한 번 등록하면
공정·일정·계약·납부일정이 한 번에
```

**서브헤드 (2줄)**
```
같은 정보를 4번 옮겨 적던 흐름을 없앴습니다.
대금은 2단계부터 6단계까지 자유롭게 나누실 수 있어요.
```

**Primary CTA**: `한 번에 등록해 보기`
**Secondary CTA**: `3분 사용 영상`

### 3-2. 베네핏 3 (아이콘 + 한 줄)

| # | 아이콘 | 헤드 | 본문 |
|---|---|---|---|
| 1 | 📋 | **한 번 입력하면 끝** | 현장명·평수·시작일만 넣으시면 공정 14개, 캘린더 일정, 계약, 납부일정이 자동으로 깔립니다. |
| 2 | 💰 | **대금 분할 자유롭게** | 계약금·착수금·중도금·잔금 기본 4단계. 2~6단계까지 사장님 손에 맞게 조정. |
| 3 | 🔄 | **한 곳에 묶여 흘러갑니다** | 일정이 밀리면 계약 납부일도 같이 밀립니다. 따로 옮겨 적지 않습니다. |

### 3-3. 사용 흐름 3스텝

**Step 1 — 기본 정보 입력 (2분)**
```
현장 이름, 주소, 평수, 시작 예정일, 등급(라이트/스탠다드/프리미엄) 정도.
공정 14개 중 안 쓰시는 건 끄셔도 됩니다.
```

**Step 2 — 자동 생성 결과 확인 (1분)**
```
- 공정 14개가 표준 순서로 깔립니다 (32평 기준 34일 일정)
- 캘린더에도 그대로 표시됩니다
- 계약·납부일정 4단계가 자동으로 생성됩니다 (계약금 10% / 착수금 40% / 중도금 40% / 잔금 10%)
```

**Step 3 — 사장님 손에 맞게 조정**
```
- 공정 일수는 드래그로 줄였다 늘렸다
- 대금 행은 +/- 로 자유롭게 2~6개
- 다 끝나고 저장만 누르시면 됩니다
```

### 3-4. FAQ

| Q | A |
|---|---|
| 이미 등록한 현장은 어떻게 되나요? | 그대로 유지됩니다. 새 현장부터 새 흐름으로 등록하실 수 있어요. |
| 공정 자동 일정이 우리 현장과 안 맞으면? | 일수·순서 모두 사장님이 바꾸실 수 있습니다. 다음 현장부터 사장님의 패턴으로 기본값 저장도 가능. |
| 대금 분할 안 쓰는 단계는 빼도 되나요? | 행 삭제로 2개까지 줄이실 수 있습니다. |
| 일정이 밀리면 납부일도 같이 미뤄지나요? | 네. 현장 시작일을 옮기면 공정·납부일정이 같이 따라옵니다. |

## 4. 카카오톡 채널 공지 초안

### 4-1. 메인 안내 (짧고 행동 유도)

```
[인테리어코치] 현장 등록이 한 번이면 됩니다

이번 업데이트로 바뀐 점.

📋 현장 한 번 등록하시면
   공정 14개·일정 캘린더·계약·납부일정 4단계가 자동으로 깔립니다.
   같은 내용 4번 옮겨 적으시던 일 사라졌습니다.

💰 대금 분할은 사장님 손에 맞게.
   계약금·착수금·중도금·잔금 기본 4단계, 2~6단계까지 자유롭게.

▸ 인테리어코치 들어가시면 첫 화면에 안내가 떠 있어요
   www.interiorcoach.co.kr
```

### 4-2. 더 짧은 버전 (리드 대상)

```
[인테리어코치 업데이트]

같은 내용 4번 옮겨 적으시던 일, 한 번에 정리되게 했습니다.
현장 등록하시면 공정·일정·계약·납부일정이 자동으로 깔려요.

대금 분할은 2단계부터 6단계까지 사장님 맞춤.

▸ www.interiorcoach.co.kr
```

### 4-3. 1주 뒤 리마인드 (안 써본 사람)

```

다음 현장 등록하실 때 한 번만 입력해 보세요.
공정 일수·대금 비율은 처음에 자동으로 깔리고,
사장님 현장 패턴에 맞게 조정하시면 됩니다.

▸ 3분 사용 영상: [링크]
▸ 1:1 도입 도움: [링크]
```

## 5. 소셜·Threads 스니펫

### 5-1. 공감 진입

```
인테리어 사장님들 현장 등록할 때
"현장 입력 → 공정 입력 → 계약 입력 → 일정 입력"
이렇게 같은 내용 4번 적으시잖아요.

이거 한 번에 정리되게 만들었습니다.
공정 14개, 캘린더, 계약, 납부일정 4단계가 한 번에 깔려요.

#인테리어코치
```

### 5-2. 대금 분할

```
대금을 2단계로 받는 사장님도 있고,
계약금·착수금·기성 1차·기성 2차·잔금 이렇게 6단계로 받는 사장님도 있어요.

사장님 손에 맞게 2~6단계 자유롭게.
기본은 계약금·착수금·중도금·잔금 4개로 깔립니다.

#인테리어코치
```

### 5-3. 짧은 한 줄

```
같은 내용 다시 안 적어도 됩니다.

현장 한 번 등록 = 공정·일정·계약·납부일정 자동.

#인테리어코치
```
