# 현장 중심 개편 — 사전 리서치

> 작성일: 2026-06-21
> 범위: ① 인테리어 업체 대금 분할 관행 ② 공종별 평균 소요일 ③ 다른 현장 관리 도구의 등록 흐름 비교

---

## 1. 대금 분할 관행

### 1-1. 가장 흔한 4단계 — 계약금 / 착수금 / 중도금 / 잔금

이 4개가 인테리어 시공 계약서에서 90% 가까이 쓰는 기본 단계입니다. 인테리어코치 기본값으로 그대로 깔아두면 됩니다.

| 단계 | 보통 비율 | 받는 시점 | 비고 |
|---|---|---|---|
| 계약금 | 10% | 계약서 작성일 | 가장 작게 받는 자리. 손해배상 기준이 되기도 함. |
| 착수금 | 30~40% | 철거 시작 전 1~3일 | 자재 발주·인부 일당 선지급 자금. |
| 중도금 | 30~40% | 공정 절반(보통 목공 마감) | 중간 결제. 진행 상황 확인 후. |
| 잔금 | 10~30% | 입주청소 끝나고 검수 후 | 하자 대비 가장 마지막에. |

**자주 쓰는 비율 조합**

| 조합 | 합계 | 어떤 업체가 자주 쓰나 |
|---|---|---|
| 10 / 40 / 40 / 10 | 100% | 자재비 부담이 큰 전체 리모델링 (가장 일반적) |
| 10 / 30 / 40 / 20 | 100% | 잔금을 조금 더 두는 보수적 업체 |
| 30 / 30 / 30 / 10 | 100% | 자재비 선납이 많은 고급 시공 |
| 10 / 50 / 30 / 10 | 100% | 자재 일괄 발주 비중이 클 때 |

### 1-2. 4단계 외에 자주 등장하는 항목

| 항목 | 언제 쓰는가 | 인테리어코치 기본값 후보 |
|---|---|---|
| **자재비** | 자재만 따로 결제받는 업체 (가구·주방·창호) | 추가 후보 ✅ |
| **선급금 / 가지급금** | 계약 전에 미리 받는 자금. 자재 단가 인상 우려 시 | 일부 업체만, 추가 후보 |
| **기성금 1차·2차·3차** | 공정 진행률(예: 30%·60%·90%)에 맞춰 청구. 큰 현장에서 주로 사용 | 추가 후보 ✅ |
| **마감금** | 잔금과 별도로 마감 공정 직전 받는 자금 | 가끔 |
| **준공금 / 인도금** | 잔금의 다른 이름 | 별도 항목 불필요 |
| **하자보수보증금** | 잔금에서 일부 유보, 6개월~1년 후 지급 | 추가 후보 |

### 1-3. 2~6단계 자유 분할 — 기본값 제안

이번 개편에서 분할 단계 수를 2~6개로 자유롭게 둘 수 있게 한 만큼, **기본 4개**로 시작하고 +/- 가능하게 하는 게 자연스럽습니다.

| 단계 수 | 추천 구성 | 어떤 현장 |
|---|---|---|
| 2 | 계약금 / 잔금 | 부분 시공·도배만 등 1주 이내 짧은 현장 |
| 3 | 계약금 / 중도금 / 잔금 | 중소 리모델링 (15평 이하) |
| **4 (기본)** | **계약금 / 착수금 / 중도금 / 잔금** | **일반 리모델링 (대부분)** |
| 5 | 계약금 / 착수금 / 자재비 / 중도금 / 잔금 | 자재 비중 큰 가구·주방 포함 시공 |
| 6 | 계약금 / 착수금 / 기성 1차 / 기성 2차 / 기성 3차 / 잔금 | 신축·대규모 리모델링 (50평 이상) |

### 1-4. 정리

- 입력 폼 기본값: **4개 행 (계약금 10% / 착수금 40% / 중도금 40% / 잔금 10%)**
- "행 추가" 버튼으로 6개까지, "삭제"로 2개까지
- 각 행의 항목 이름은 자유 입력(기본 4개 외에 "자재비", "기성 1차" 등 자주 쓰는 라벨을 자동완성 제안)
- 비율 합계가 100%가 안 되면 저장은 되지만 경고 표시

## 2. 공종별 평균 소요일 (32평 아파트 전체 리모델링 기준)

일정 자동 생성의 기본값으로 쓸 일수입니다. 면적·등급에 따라 ±1~2일 차이는 자연스럽게 발생.

### 2-1. 공종별 표준 일수

| 공종 | 평균 일수 | 범위 | 비고 |
|---|---|---|---|
| **철거** | 2일 | 1~3일 | 부분 철거는 1일, 전체는 2~3일 |
| **설비·배관** | 2일 | 1~3일 | 욕실·주방 위치 변경 시 길어짐 |
| **전기** | 3일 | 2~4일 | 배선 + 분전반 + 등기구 박스 |
| **샷시·창호** | 3일 | 2~4일 | 외부 자재 도착 일정에 좌우 |
| **목공** | 7일 | 5~10일 | **가장 길고 중요한 공정.** 천장·우물천장·간접등 박스 등 |
| **타일** | 5일 | 4~7일 | 욕실·주방·현관. 줄눈까지. |
| **도장·필름** | 4일 | 3~5일 | 페인트 건조 시간 포함 |
| **도배** | 3일 | 2~3일 | 천장·벽지·풀칠 건조 |
| **바닥재** | 2일 | 1~3일 | 마루는 1일, 타일 바닥은 2~3일 |
| **주방 설치** | 1일 | 1~2일 | 자재 입고 후 당일 가능 |
| **가구·붙박이장** | 2일 | 1~3일 | 실측 후 제작 일정 별도 |
| **욕실 설치** | 3일 | 2~4일 | 양변기·세면대·샤워부스·악세사리 |
| **도어·중문** | 2일 | 1~3일 | 현관·방문·중문. 제작 일정 별도 |
| **조명** | 1일 | 1~2일 | 등기구 설치 + 조명 점검 |
| **입주청소** | 1일 | 1일 | |

### 2-2. 공정 순서 (병렬 가능 작업 표시)

순차로 진행되는 게 원칙이지만, 일부 공종은 같이 돌릴 수 있습니다.

```
1순위: 철거
2순위: 설비·배관 / 샷시 (병렬 가능)
3순위: 전기 (목공 들어가기 전에 배선 마무리)
4순위: 목공 (가장 길고, 다른 공정의 기준이 됨)
5순위: 타일 (욕실 / 주방 / 현관 동시 진행 가능)
6순위: 도장·필름
7순위: 도배
8순위: 바닥재
9순위: 주방 / 욕실 / 가구 설치 (병렬 가능)
10순위: 도어·중문
11순위: 조명
12순위: 입주청소
```

### 2-3. 표준 일정 자동 생성 — 32평 기본값

위 순서대로 병렬 가능한 공종을 묶으면 **약 4주 6일 (34일)** 표준 일정이 나옵니다.

| 주차 | 진행 공종 | 누적 일수 |
|---|---|---|
| 1주차 | 철거 → 설비·전기 → 샷시 | 1~7일 |
| 2주차 | 목공 시작 | 8~14일 |
| 3주차 | 목공 마감 → 타일 | 15~21일 |
| 4주차 | 도장·필름 → 도배 → 바닥재 | 22~28일 |
| 5주차 | 주방·욕실·가구·도어·조명·청소 | 29~34일 |

- 인테리어코치는 이 패턴을 기본값으로 깔고, 사장님이 +/- 조정.
- 외부 자재 입고 지연 / 휴일 / 협력업체 일정에 따라 ±5일 변동 자연스러움.

### 2-4. 면적 보정

| 평수 | 곱하기 | 결과 |
|---|---|---|
| ~20평 | × 0.7 | 약 24일 |
| 21~30평 | × 0.9 | 약 31일 |
| **31~40평 (기준)** | × 1.0 | **약 34일** |
| 41~50평 | × 1.15 | 약 39일 |
| 51평~ | × 1.3 | 약 44일 |

목공·도배·바닥재처럼 면적 영향이 큰 공종만 보정, 짧은 공종(조명·청소)은 거의 동일.

## 3. 다른 현장 관리 도구의 등록 흐름 비교

### 3-1. 인테리어코치 (현재)

| 단계 | 입력 |
|---|---|
| 1 | 현장 등록 (이름·주소·평수·고객) |
| 2 | 계약 따로 등록 (계약 금액·납부일정) |
| 3 | 공정 따로 등록 (공종별 시작/종료일) |
| 4 | 일정 캘린더에 따로 표시 |

→ **같은 정보(현장명·일정·금액)를 4번 입력.** 이번 개편의 핵심이 이 반복을 없애는 것.

### 3-2. 현장의신 (국내)

| 단계 | 입력 |
|---|---|
| 1 | 현장 + 고객 기본 정보 |
| 2 | 공정 14개 미리 등록되어 있고 켜기/끄기 |
| 3 | 계약·결제는 별도 모듈 |

→ 공정은 자동 깔리지만 계약·일정은 별도 입력.

### 3-3. 빌드온 (국내)

| 단계 | 입력 |
|---|---|
| 1 | 현장 + 일정 동시 등록 (간트 자동 생성) |
| 2 | 계약·수금은 별도 |

→ 일정은 자동, 계약은 따로. 비용은 월 10만원대.

### 3-4. Buildertrend (해외)

| 단계 | 입력 |
|---|---|
| 1 | 프로젝트 등록 (이름·주소·예산) |
| 2 | 템플릿 선택 → 공정·결제 스케줄·체크리스트 일괄 생성 |
| 3 | 고객 포털 자동 발급 |

→ **템플릿 한 번 골라 모든 게 자동.** 가장 가까운 모델. 가격 $499/월.

### 3-5. Houzz Pro (해외)

| 단계 | 입력 |
|---|---|
| 1 | 프로젝트 + 고객 |
| 2 | 결제 마일스톤 (Payment Schedule) 자동 제안 |
| 3 | 일정은 별도 |

→ 결제 자동, 일정 따로. 가격 $99/월부터.

### 3-6. 정리

- 국내 도구는 일정·계약·결제가 **분리 입력**되어 있고, 해외 도구(Buildertrend)는 **템플릿 하나로 일괄 생성**합니다.
- 이번 개편의 목표는 Buildertrend 모델 — **현장 한 번 등록 = 공정·일정·계약·납부일정 자동 생성**.
- 한국 인테리어 업체 워크플로우(공종 분류·계약금/착수금/중도금/잔금 4단계·반장·기성 등)에 맞춤.

## 4. 종합 결정

| 결정 | 값 |
|---|---|
| 대금 분할 기본값 | 4단계 (계약금 10% / 착수금 40% / 중도금 40% / 잔금 10%) |
| 대금 분할 범위 | 2~6단계, 자유 추가/삭제 |
| 항목 라벨 자동완성 | 계약금·착수금·중도금·잔금·자재비·기성 1차/2차/3차·하자보수보증금 |
| 일정 기본값 | 32평 기준 34일, 14개 공종 표준 순서 |
| 면적 보정 | × 0.7~1.3 (면적 구간별) |
| 현장 등록 → 자동 생성 항목 | 공정 14개 + 일정 캘린더 + 계약 + 납부일정 4행 |
