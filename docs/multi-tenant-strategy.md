### 1-D. 벤치마킹 핵심 인사이트

1. **한국 시장에 인테리어 팀 협업 SaaS가 사실상 부재** — 집닥/오늘의집은 마켓플레이스, 아키스케치는 디자인 툴. 팀 프로젝트 관리 SaaS는 블루오션.
2. **Monday.com의 "무료 Viewer" 전략** — 고객(시공주)에게 읽기 전용 접근 제공 = 강력한 lock-in + 바이럴.
3. **Buildertrend의 "무제한 유저" 모델** — 높은 기본가($499+)로 유저 수 걱정 없이 팀 전체 도입 유도.
4. **Notion의 부드러운 전환** — 개인 사용 → 초대 한 번으로 팀 전환. 이탈 최소화의 교과서.

## 산출물 2. 가격 전략 재설계

### 2-A. 한국 인테리어 업체 인원 구성 현실

| 규모 | 인원 | 비율 | 전형적 역할 |
|------|------|------|------------|
| 1인 사업자 | 1명 | ~27% | 대표 (설계+영업+현장 겸직) |
| 소규모 | 2-5명 | ~46% | 대표 + 디자이너 + 현장소장 + 경리 |
| 중규모 | 6-15명 | ~18% | 디자인팀 + 시공관리 + 영업 + 관리부 |
| 대규모 | 16명+ | ~9% | 부서별 분화, 프랜차이즈 |

> **핵심**: 전체 시장의 73%가 5인 이하. "5인 포함"은 시장의 73%를 커버하는 적절한 수치.
> 다만 현재 Pro 플랜(₩299,000)에서만 5인을 제공하는 것은 가격 장벽이 높음.

### 2-B. 한국 인테리어 SMB의 SaaS 지불 의사

- 월 SaaS 지출 Sweet Spot: **₩30,000-100,000**
- 디지털 도구 채택률: **5-10%** (카카오톡이 사실상 업무 도구)
- Per-seat 과금에 대한 저항: **높음** ("경리가 거의 안 쓰는데 왜 같은 돈?")
- Per-workspace 선호: **높음** (소규모 팀, 역할별 사용 빈도 차이 큼)

### 2-C. 가격 시나리오 3가지

#### 시나리오 A: 워크스페이스 기반 계단식 (추천)

| 플랜 | 월 가격 | 연간(20%↓) | 포함 인원 | 추가 시트 | 핵심 기능 |
|------|---------|-----------|----------|----------|----------|
| **Free** | ₩0 | - | 1명 | - | 견적코치 기본, 현장 3개, 고객 20명 |
| **Team** | ₩79,000 | ₩63,200 | 5명 | ₩9,900/인 | 전 기능, 무제한 현장, 마케팅 기본 |
| **Business** | ₩199,000 | ₩159,200 | 15명 | ₩7,900/인 | + 고급 보고서, 전자계약, 마케팅 전체 |
| **Enterprise** | ₩399,000 | ₩319,200 | 무제한 | - | + 멀티지점, API, 전담 매니저 |

- **장점**: 한국 SMB sweet spot(₩79,000)에 팀 기능 제공, 가격 장벽 대폭 낮춤
- **단점**: 현재 대비 ARPU 하락 가능성 (Pro ₩299K → Team ₩79K)
- **예상 ARPU**: ₩85,000-120,000/월 (볼륨 증가로 보상)
- **적합**: Product-Led Growth 전략, 시장 침투 우선

#### 시나리오 B: Per-Seat + 기본 워크스페이스 (하이브리드)

| 플랜 | 월 기본가 | 기본 시트 | 추가 시트 | 뷰어(무료) | 핵심 기능 |
|------|----------|----------|----------|-----------|----------|
| **Free** | ₩0 | 1 | - | - | 견적코치 기본, 현장 3개 |
| **Starter** | ₩49,000 | 1 | ₩29,000/인 | 3명 | 전 기능, 무제한 현장 |
| **Pro** | ₩149,000 | 3 | ₩29,000/인 | 무제한 | + 마케팅, 전자계약, 고급 보고 |
| **Enterprise** | ₩399,000 | 10 | ₩19,000/인 | 무제한 | + 멀티지점, API, 전담 매니저 |

- **장점**: 수익이 팀 성장과 직결, 1인도 ₩49,000에 유료 전환 가능
- **단점**: Per-seat 저항감, 계정 공유 유발 가능
- **예상 ARPU**: ₩110,000-180,000/월
- **적합**: 수익 최적화 우선, 글로벌 확장 고려 시

#### 시나리오 C: 현재 구조 유지 + 팀 기능 Add-on

| 플랜 | 월 가격 | 인원 | 팀 Add-on | 핵심 변경 |
|------|---------|------|----------|----------|
| **Free** | ₩0 | 1명 | - | 변경 없음 |
| **Starter** | ₩149,000 | 1명 | +₩29,000/인 | 추가 시트 옵션 신설 |
| **Pro** | ₩299,000 | 5명 | +₩29,000/인 | 변경 없음 |
| **Enterprise** | ₩599,000 | 무제한 | 포함 | 변경 없음 |

- **장점**: 기존 가격 체계 유지, 혼란 최소화
- **단점**: Team 기본가(₩149K)가 여전히 높아 소규모 업체 진입 장벽
- **예상 ARPU**: ₩160,000-220,000/월
- **적합**: 리스크 최소화, 점진적 전환

### 2-D. 추천 의견

**시나리오 A (워크스페이스 기반 계단식)를 추천합니다.**

근거:
1. 한국 인테리어 시장은 디지털 도구 채택률이 5-10%로 극히 낮음 → **가격 장벽을 최대한 낮춰야** 시장 침투 가능
2. 전체 시장의 73%가 5인 이하 → **₩79,000에 5인** 제공이 대부분의 업체를 커버
3. 경쟁자(집닥, 오늘의집)가 수수료 모델 → SaaS 구독 모델의 가치를 증명하려면 **낮은 진입 가격** 필수
4. "무료 뷰어" 시트 도입으로 고객(시공주)/하도급에 읽기 전용 접근 → **바이럴 + lock-in** 효과
5. ARPU 하락은 **유료 전환율 상승(×3-5배)**으로 충분히 보상 가능

### 2-E. 무료 플랜 팀 기능 오픈 범위 (유료 전환 유도)
