### 1-3. 핵심 발견

> **한국 시장에 "소비자가 직접 만드는 AI 공정표" 서비스는 없다.**

- 카스웍스: 유일한 전문 공정관리 SaaS이지만 **시공사 전용** (B2B)
- 오늘의집/집닥: 매칭·견적까지만. 공정 일정은 시공사가 알아서 관리
- 해외(Buildertrend 등): 고객 포탈이 있으나 "시공사가 만든 것을 보여주는" 수준
- **인테리어코치 AI 공정매니저는 최초의 "소비자 주도 공정표 생성 도구"**

### 1-4. 차별화 포지셔닝

```
기존 구조:
  시공사 → (내부 관리) → 고객에게 "4주 걸립니다" 통보
  소비자: 근거 없이 믿거나, 의심만 하거나

인테리어코치 구조:
  소비자 → AI 공정표 직접 생성 → 시공사 일정 크로스체크
  "시공사 말만 믿지 마세요. AI로 직접 확인하세요"
```

**핵심 차별점:**

| 차별점 | 설명 |
|--------|------|
| 소비자 주도 | 시공사가 아닌 고객이 직접 공정표 생성 |
| AI 의존성 체크 | 11개 의존성 규칙으로 공종 순서 오류 자동 탐지 |
| 실시간 비용 연동 | 평수·시즌·공종별 비용 범위 자동 산출 |
| 견적코치 시너지 | 견적 분석 → 공정 검증 → 품질 체크 원스톱 |
| 프리미엄 퍼널 | 무료(공정표) → 유료(비용분석·발주·품질) → 코디네이터 |

### 1-5. 견적코치 ↔ 인테리어코치 시너지 분석

```
[견적코치 시뮬레이터]          [AI 공정매니저]
  비용이 적정한가? ────────→ 공정 순서가 맞는가?
  항목별 단가 확인  ←──────── 공종별 비용 분석

  Cross-Sell 포인트:
  ① 견적 분석 완료 → "이 공종 순서, AI로 검증해보세요" CTA
  ② 공정표 비용 탭 → "이 비용이 적정한지 견적코치에서 확인" CTA (이미 구현)
  ③ 공정표 완성 → "실제 견적서와 비교해보세요" CTA
```

- 타깃 키워드: 인테리어 체크리스트, 인테리어 품질 점검, 인테리어 하자 체크
- 구조:
  - 공종별 품질 체크포인트 전체 리스트 (TRADES의 qualityCheck 데이터 활용)
  - 단계별 필수 확인 사항
  - 사진 촬영 필수 시점
  - CTA: "AI가 공종별 체크리스트를 자동 생성 → [링크]"
- 예상 분량: 3,500~4,000자

### 2-4. 인스타그램/릴스 콘텐츠

#### 릴스 (15~30초)
1. **화면 녹화**: AI 공정매니저에서 공정표 만드는 과정 (평수 선택 → 공종 선택 → 생성 → 간트차트 애니메이션)
2. **Before vs After**: 엑셀 공정표(복잡, 수동) → AI 공정표(자동, 시각화) 트랜지션
3. **의존성 경고 데모**: "방수 안 넣으면?" → 빨간 경고 뜨는 화면 녹화

#### 카드뉴스 시리즈: "알면 아끼는 인테리어 팁"
- 카드 1: 공종 아이콘 + 이름
- 카드 2: 평균 비용 (평당)
- 카드 3: 절약 팁 (savingTip 데이터)
- 카드 4: 품질 체크포인트
- 카드 5: CTA → 인테리어코치

### 3-2. 전환 포인트 분석

#### 무료 → 스탠다드 전환 트리거

| Blur Lock 위치 | 노출 메시지 | 심리 트리거 | 예상 클릭률 |
|---------------|-----------|-----------|-----------|
| **비용분석 탭** | "공종별 상세 비용, 절약 팁 확인" | 절약 동기 ("이만큼 아낄 수 있는데...") | 15~25% |
| **발주·준비 탭** | "발주 타이밍·자재 리스트 확인" | 불안감 ("자재 납기 늦으면 공사 지연") | 10~15% |
| **품질체크 탭** | "품질 체크포인트·하자 방지" | 공포 ("하자 나면 재시공 비용이...") | 20~30% |

**예상 전환율:**
- 공정표 완성(Step 3) → Blur Lock 1회 이상 클릭: 40~60%
- Blur Lock 클릭 → 패키지 탭 확인: 60~70%
- 패키지 탭 → 스탠다드 결제: 5~10%
- **종합 전환율 (무료 → 스탠다드): 1.5~4%**

#### 견적코치 → 인테리어코치 크로스셀

### 5-1. 핵심 카피 후보 (A/B 테스트용)

| # | 카피 | 트리거 | 테스트 위치 |

| B | **"시공사 일정표, 진짜 맞는 건지 AI로 확인"** | 불신/검증 | Threads, 인스타 |
| C | **"18개 공종, 의존성, 비용까지 — 공정표 자동 생성"** | 전문성 | 네이버 블로그 |
| D | **"인테리어 공사, 순서가 틀리면 돈이 새요"** | 공포/절약 | Threads 티저 |
| E | **"방수 빠진 공정표? AI가 잡아줍니다"** | 구체적 공포 | 인스타 릴스 |

**A/B 테스트 계획:**
1. **Week 1-2**: A vs B — Threads 2개 포스트에서 참여율 비교
2. **Week 2-3**: C vs D — 블로그 제목에서 CTR 비교
3. **Week 3-4**: 승자 조합으로 인스타 광고 소재 제작

### 5-2. 톤 앤 매너

```
DO:
  ✅ "~하면 이만큼 아낄 수 있어요" (절약 동기)
  ✅ "AI가 자동으로 체크합니다" (편의성)
  ✅ "시공사 견적서와 비교해보세요" (고객 편)
  ✅ 구체적 숫자: "11개 의존성 규칙", "50+ 품질 체크"
  ✅ Before/After 비교 (엑셀 vs AI)

DON'T:
  ❌ "~해야 합니다" (강압)
  ❌ 시공사를 직접 비난 (갈등 유발)
  ❌ 과장된 절약 약속
  ❌ 전문 용어 남발 (방수턴업, 분배기 등 → 풀어서 설명)
```

**공포+절약 조합 공식:**
```
"[하자 사례/리스크] + [구체적 금액] + [AI 해결책]"

예시:
"방수 빠진 공정표로 시공하면 욕실 재시공 300~500만원.
 AI가 의존성을 체크하면 3분 만에 예방됩니다."
```
