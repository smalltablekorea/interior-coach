# 견적코치 마케팅 센터 — 브랜드 톤 가이드

> 최종 수정: 2026-03-21

---

## 1. 브랜드 핵심 정체성

### 견적코치가 하는 일

> 인테리어 견적서를 AI로 분석하여, 건축주에게 **시세 기반의 객관적 판단 근거**를 제공하는 서비스.

### 한 줄 가치 제안

> "받으신 견적서가 적정한지, 데이터로 확인해 드립니다."

### 핵심 키워드 (태그라인 후보)

| 순위 | 키워드 | 이유 |
|------|--------|------|
| 1 | **시세 비교** | 객관적 데이터 기반 (가장 강력한 USP) |
| 2 | **절감 근거** | 구체적 금액 절약 가능성 |
| 3 | **협상 도움** | 업체와 대등한 대화를 할 수 있게 |
| 4 | **객관적 분석** | 업체 편도 건축주 편도 아닌 중립적 입장 |

## 2. 톤 앤 매너 원칙

### DO (이렇게 써라)

| 원칙 | 예시 |
|------|------|
| **신뢰감 있는 정보 제공** | "시세 데이터 기반으로 분석합니다" |
| **구체적인 수치 활용** | "평균 340만원 절감 포인트 발견" |
| **객관적/중립적 표현** | "적정가인지 확인해 드립니다" |
| **건축주 입장에서 실용적** | "업체 미팅 전에 확인하세요" |
| **존댓말, 정중하지만 친근** | "고객님, ~해 보세요" |
| **행동 유도는 부드럽게** | "필요하실 때 확인해 주세요" |
| **공종/자재 전문 용어 적절히** | "목공사 시세", "합지 vs 실크" |
| **절약 가능성은 근거와 함께** | "타일 단가가 시세 대비 22% 높아요" |

### DON'T (이렇게 쓰지 마라)

| 금지 표현 | 이유 | 대안 |
|-----------|------|------|
| "최저가 보장" | 과대 광고 | "시세 대비 적정가 분석" |
| "업체 바가지 조심!" | 공격적/선정적 | "과다 청구 가능성 확인" |
| "지금 안 하면 손해" | 공포 마케팅 | "확인해 두시면 도움이 됩니다" |
| "XX% 할인 마감 임박!!" | 긴급성 남발 | 사용 빈도 최소화, 실제 기한만 |
| "무조건 깎으세요" | 무책임한 조언 | "시세 근거를 제시하면 합리적 대화 가능" |
| "업체를 믿지 마세요" | 업계 적대적 | "객관적 비교로 판단하세요" |
| "인테리어 사기 당하지 마세요" | 선정적 | "견적서 주요 확인 포인트" |
| "100% 정확한 분석" | 과대 약속 | "시세 데이터 기반 분석" |
| "Lorem ipsum", "Test" | 더미 문구 | 견적코치 도메인 실제 문구 |

## 3. 보이스 페르소나

### 견적코치의 인격

> **경험 많은 친구의 시공 전문가 지인** 같은 느낌.
>
> - 10년차 인테리어 전문가
> - 건축주 편이지만 업체를 적대시하지 않음
> - 데이터와 시세를 기반으로 조언함
> - 어려운 용어는 쉽게 풀어서 설명
> - "이건 시세상 좀 높아 보여요. 이유가 있을 수 있으니 업체에 확인해 보세요."

### 보이스 스펙트럼

```
격식 ←————————————————→ 캐주얼
         ●
     (약간 정중, 친근)

전문적 ←————————————————→ 일반적
              ●
        (적절히 전문)

단정적 ←————————————————→ 제안적
                 ●
          (근거 제시 + 제안)
```

## 4. 표현 가이드 (카테고리별)

### 가격/비용 관련

| 상황 | 좋은 표현 | 나쁜 표현 |
|------|-----------|-----------|
| 비쌀 때 | "시세 평균 대비 15% 높습니다" | "바가지입니다" |
| 쌀 때 | "시세 평균 대비 낮은 편이에요. 자재 등급을 확인해 보세요" | "싼 게 비지떡" |
| 적정할 때 | "시세 범위 내에서 적정한 수준입니다" | "괜찮은 것 같아요" |
| 절감 가능 | "이 항목에서 약 120만원 절감 여지가 있습니다" | "120만원 아끼세요!!" |

### 업체 관련

| 상황 | 좋은 표현 | 나쁜 표현 |
|------|-----------|-----------|
| 비교 | "업체별 견적을 객관적으로 비교할 수 있습니다" | "어떤 업체가 사기인지 알 수 있습니다" |
| 추천 | "분석 결과에 맞는 추천 업체를 연결해 드려요" | "이 업체가 최고입니다" |
| 협상 | "시세 근거를 제시하면 합리적인 대화가 가능합니다" | "이걸 보여주고 깎으세요" |

### 행동 유도 (CTA)

| 상황 | 1순위 표현 | 2순위 표현 | 금지 |
|------|-----------|-----------|------|
| 가입 | "무료로 시작하기" | "견적 분석 시작" | "지금 가입!!" |
| 업로드 | "견적서 올리기" | "업로드 시작하기" | "지금 바로!" |
| 결제 | "전체 리포트 확인하기" | "리포트 보기" | "결제하기" (직접적 금전 표현 피함) |
| 문의 | "업체에 문의하기" | "추천 업체 보기" | "빨리 연락하세요" |
| 후기 | "후기 남기기" | "의견 남기기" | "후기 필수!" |

### 시간 표현

| 상황 | 좋은 표현 |
|------|-----------|
| 분석 소요 | "약 3분 이내" |
| 리포트 유효기간 | "30일간 열람 가능" |
| 후기 작성 시간 | "30초면 충분해요" |
| 업로드 안내 | "사진 한 장이면 됩니다" |

## 5. 숫자/통계 사용 규칙

### 사용해도 좋은 통계

| 통계 | 조건 |
|------|------|
| "평균 340만원 절감 포인트 발견" | 실제 분석 결과 기반, 주기적 업데이트 |
| "최근 30일 1,247건 분석" | 실시간 데이터 기반 |
| "이용 고객 87% 만족" | 실제 설문 결과 있을 때만 |
| "17개 공종 분석" | 시스템 사실 |

### 사용하면 안 되는 통계

| 통계 | 이유 |
|------|------|
| "100% 정확" | 과대 약속 |
| "무조건 절약" | 보장 불가 |
| "업체 대비 50% 저렴" | 비교 근거 불명확 |
| 오래된 수치 (3개월 이상) | 신뢰도 하락 |

### 숫자 표기 규칙

```
금액: 340만원 (만원 단위), 1억 2,000만원 (억 단위)
비율: 15% (정수), 22.5% (소수 1자리까지)
건수: 1,247건 (천 단위 콤마)
기간: 3분, 30초, 24시간, 7일, 30일
순서: ①②③ 또는 1. 2. 3.
```

## 6. 채널별 톤 조정

### 검색광고 (네이버/구글)

- **초 간결** — 25자 이내 핵심 + 구체적 숫자
- "인테리어 견적 AI 분석" > "인테리어 견적 분석 서비스"
- CTA는 동사로: "시작하기", "확인하기", "비교하기"

### 이메일

- **정보 밀도 높게** — 구체적 수치, 사례, 단계별 안내
- 제목은 궁금증 유발: "견적 분석 리포트가 대기 중이에요"
- 본문은 짧은 문단 + 볼드 핵심 + 명확한 CTA 버튼

### 카카오 알림톡

- **간결 + 핵심** — 3~5줄 본문 + 링크
- "[견적코치]" 발신자 표시 필수
- 핵심 가치 1개만 전달
- 버튼 텍스트는 동사형: "▶ 업로드 시작하기"

### 블로그/콘텐츠

- **교육적 + 실용적** — 실제 사례, 체크리스트, 비교표
- SEO 키워드 자연스럽게 포함
- 하단 CTA는 콘텐츠와 자연스럽게 연결

### SMS

- **최소 정보** — 핵심 메시지 + URL만
- 20자 이내 핵심 문구 + 링크
- 수신 거부 안내 필수

## 8. 시즌/상황별 톤 조정

| 시즌/상황 | 톤 조정 |
|-----------|---------|
| 봄 이사철 (3~5월) | 약간 긴급 + 사례 중심 — "이사 준비 시작하셨나요?" |
| 가을 인테리어 시즌 (9~11월) | 트렌드 + 정보 — "올해 인테리어 트렌드와 비용" |
| 연말 (12월) | 정산 + 계획 — "내년 인테리어 계획, 미리 견적 비교해 보세요" |
| 경기 침체기 | 절약 강조 — "불필요한 지출을 줄이는 방법" |
| 자재가 급등기 | 시세 변동 안내 — "최근 자재가 변동과 견적 확인 포인트" |
| 서비스 장애 | 투명한 안내 — "일시적 지연이 발생하고 있습니다. 빠르게 복구 중입니다." |

## 9. 톤 체크리스트 (카피 작성 시 확인)

카피를 작성한 후 아래 체크리스트로 검증한다.

- [ ] 과장/허위 표현이 없는가?
- [ ] 구체적인 숫자/근거가 포함되어 있는가?
- [ ] 공포 마케팅이나 선정적 표현이 없는가?
- [ ] 업체를 적대시하는 표현이 없는가?
- [ ] CTA가 명확하고 부드러운가?
- [ ] 견적코치 보이스 (전문적 + 친근)에 맞는가?
- [ ] 개인정보/민감 데이터 노출이 없는가?
- [ ] 한국어 맞춤법이 정확한가?
- [ ] 채널별 글자수 제한을 준수하는가?
- [ ] 수신 거부/개인정보 안내가 필요한 채널인가?
