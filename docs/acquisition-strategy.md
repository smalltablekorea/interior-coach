### 2. 네이버 카페/커뮤니티

| 항목 | 내용 |
|------|------|
| **타겟** | 아파트 입주 예정자, 인테리어 관심 소비자 → 견적코치 유입 → 업체 연결 |
| **예상 CAC** | ₩5,000~10,000 |
| **주간 액션** | 견적코치 무료 분석 후기/정보성 콘텐츠 5건/주 |
| **KPI** | 견적코치 분석 요청 수, 클릭률 |

#### 타겟 카페 리스트

| 카페 | 회원수 | 공략 방법 |
|------|--------|----------|
| 인테리어 이야기 | 100만+ | 정보성 포스팅 (견적 비교 방법, 비용 절감 팁) |
| 부동산 스터디 | 80만+ | 투자 관점의 인테리어 가치 상승 콘텐츠 |
| 아파트 입주예정자 카페 (단지별) | 수천~수만 | **핵심**: 입주 시점 타겟, 견적코치 무료 분석 안내 |
| 셀프 인테리어 카페 | 50만+ | 비용 비교 도구 소개 |
| 맘카페 (맘스홀릭 등) | 200만+ | 육아 공간 인테리어 + 견적 체크 콘텐츠 |

#### 4주 로드맵

| 주차 | 액션 | 목표 |
|------|------|------|
| 1주 | 카페 5곳 가입, 커뮤니티 규칙 파악, 계정 활동 시작 | 카페 5곳 활성 |
| 2주 | 정보성 콘텐츠 3건 발행 ("인테리어 견적 받았을 때 확인해야 할 5가지") | 조회수 1,000+ |
| 3주 | 견적코치 체험 후기 형태 콘텐츠 5건 + 입주카페 2곳 진입 | 견적코치 유입 50건 |
| 4주 | A/B 테스트 — 후기형 vs 정보형 vs 질문형 콘텐츠 비교 | 최적 포맷 도출 |

### 3. 인스타그램 스토리 광고

| 항목 | 내용 |
|------|------|
| **타겟** | 인테리어 업체 대표 (B2B) + 인테리어 관심 소비자 (B2C) |
| **예상 CAC** | ₩15,000~30,000 (리드 기준) |
| **주간 액션** | 크리에이티브 3종 + A/B 테스트 |
| **KPI** | CTR, CPC, 회원가입 전환율 |

#### 크리에이티브 전략

| 크리에이티브 | 타겟 | 훅 카피 | CTA |
|------------|------|---------|-----|
| A. 비용 절감 | 업체 대표 | "블로그 대행 70만원 → 0원으로 만드는 법" | 무료 시작하기 |
| B. 견적 분석 | 소비자 | "인테리어 견적, 바가지인지 3초 만에 확인" | 무료 분석받기 |
| C. 비포/애프터 | 소비자 | 시공 사례 캐러셀 + "이 견적이 적정가인지 궁금하다면" | 견적코치 체험 |

#### 타겟팅 설정

| 세그먼트 | 관심사 | 연령 | 지역 |
|---------|--------|------|------|
| 업체 대표 | 인테리어 시공, 건축, 소상공인, 자영업 | 30~55세 | 수도권 + 광역시 |
| 소비자 | 인테리어, 홈데코, 오늘의집, 이사, 신혼 | 25~45세 | 전국 |

#### 4주 로드맵

| 주차 | 액션 | 예산 | 목표 |
|------|------|------|------|
| 1주 | 크리에이티브 3종 제작, 광고 셋업, 소규모 테스트 시작 | 25만 원 | 데이터 수집 |
| 2주 | A/B 테스트 — 크리에이티브별 CTR/CPC 비교 | 25만 원 | 최적 크리에이티브 선별 |
| 3주 | 승리 크리에이티브에 예산 집중, 유사타겟 테스트 | 50만 원 | CPA 최적화 |
| 4주 | 스케일업 또는 새 크리에이티브 추가 | 50만 원 | 월 150만원 기준 CAC 확정 |

**월 예산**: 150만 원 (테스트 기간)

### 4. 부동산 중개소 파트너십

| 항목 | 내용 |
|------|------|
| **타겟** | 이사/신축 고객 (중개소 경유) |
| **예상 CAC** | ₩20,000~40,000 |
| **주간 액션** | 중개소 10곳 제안서 발송 |
| **KPI** | 파트너 중개소 수, 리퍼럴 전환 |

#### 파트너십 모델

```
고객이 중개소에서 매매/임대 계약
        ↓
중개소가 "인테리어 견적 분석" 안내 (QR코드 or 카드)
        ↓
고객이 견적코치에서 무료 분석
        ↓
견적코치 → 인테리어 업체 연결
        ↓
중개소에 리퍼럴 수수료 지급 (건당 2~5만원)
```

#### 제안서 핵심 내용
- 중개소 혜택: 부가 서비스로 고객 만족도 상승, 리퍼럴 수수료 수익
- 도입 방법: QR코드 카드 10장 비치, 앱 설치 불필요
- 비용: 중개소 측 0원

#### 4주 로드맵

| 주차 | 액션 | 목표 |
|------|------|------|
| 1주 | 제안서 작성, QR카드 디자인, 인천 지역 중개소 리스트업 30곳 | 제안서 완성 |
| 2주 | 인천 지역 10곳 방문/제안 | 파트너 3곳 |
| 3주 | 서울 강남/송파/마포 10곳 방문/제안 | 파트너 5곳 |
| 4주 | 성과 측정, 리퍼럴 트래킹 셋업 | 전환 데이터 확보 |

### 5. 오늘의집/숨고 리스팅

| 항목 | 내용 |
|------|------|
| **타겟** | 인테리어 수요자 (소비자) |
| **예상 CAC** | ₩30,000~50,000 |
| **주간 액션** | 리스팅 등록 + 리뷰 확보 |
| **KPI** | 리스팅 노출수, 문의 전환 |

#### 전략
- 자체 3개 브랜드(Small Table / Green Avenue / 이매동신)로 오늘의집/숨고에 입점
- 시공 포트폴리오에 **인테리어코치 견적 분석 결과** 첨부 → 차별화
- 시공 고객에게 인테리어코치 견적코치 무료 분석 제공 → 리뷰 유도
- 리뷰 10건 이상 확보 후 추천 노출 목표

#### 4주 로드맵

| 주차 | 액션 | 목표 |
|------|------|------|
| 1주 | 오늘의집 전문가 시공 입점 신청 (3개 브랜드) | 심사 접수 |
| 2주 | 숨고 고수 등록, 포트폴리오 업로드 (10건 이상) | 등록 완료 |
| 3주 | 기존 시공 고객에게 리뷰 요청 (5건) | 리뷰 5건 |
| 4주 | 숨고 견적 응답 시 견적코치 분석 첨부, 리뷰 추가 확보 | 리뷰 10건 |
