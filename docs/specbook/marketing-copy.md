### 2-3. 사용 흐름 3스텝

**Step 1 — 카탈로그 만들기**
```
공종별로 옵션 2~3개를 골라 담습니다. 자재 사진·치수·단가는 자재 DB에서 자동.
공용 템플릿이나 다른 현장 복제로 시작하면 더 빠릅니다.
```

**Step 2 — 고객에게 링크 전달**
```
카탈로그 페이지에서 링크 발급. 카톡으로 한 번 보내면 끝.
고객은 회원가입 없이 링크만으로 접근합니다.
```

**Step 3 — 제출 받기, 한 화면에서 확인**
```
고객이 공종별로 자재를 골라 제출하면 사장님 페이지에 그대로 정리됩니다.
확정된 자재는 자재 발주서·견적서·세금계산서로 한 번에 흘러갑니다.
```

| Q | A |
|---|---|

## 4. 1:1 데모·세일즈 스크립트 (2~3분)

### 4-1. 오프닝 (15초)

```
사장님, 마감재 카탈로그 보통 어떻게 정리하세요?
혹시 엑셀로 매번 새로 짜시거나, 카톡으로 사진 보내고
답 받으시지 않나요?
```

→ 거의 모든 사장님이 "맞아요" 하고 답함. **이 답이 다음 문장 진입 신호.**

### 4-2. 문제 공감 (30초)

```
저희가 인테리어 업체 사장님들 100분 넘게 인터뷰했더니,
이 두 가지가 가장 큰 시간 도둑이더라고요.

하나는 카탈로그 매번 새로 만드는 거고요,
다른 하나는 고객이 카톡에서 결정 못 내려서 일주일 늘어지는 거예요.

이거 두 개 때문에 누락도 생기고, 시공 일정도 밀리고요.
```

### 4-3. 솔루션 데모 (1분)

화면 공유 / 노트북 보여주기:

```
이게 스펙북이에요. 보시는 것처럼 공종별로 카탈로그를 만들고요,
(템플릿 클릭) 다른 현장에서 만들어 둔 거 복제도 됩니다.

여기 링크 발급을 누르시면, 이 링크 하나만 고객한테 카톡으로 보내시면 돼요.
고객은 회원가입 같은 거 없이 바로 이 화면이 열려요.
(고객 화면 미리보기)

사진·치수·옵션이 한 화면에 있어서 고객이 결정하기 편하고,
사장님은 이 페이지(돌아옴) 하나로 누가 뭐 골랐는지 다 보세요.
```

### 4-4. 가치 정리 (30초)

```
사장님이 매번 들이시던 카탈로그 만드는 시간이 1/10 정도로 줄고요,
고객 답 받는 시간이 일주일에서 하루 이틀로 줄어드는 패턴을
저희 베타 사용 업체에서 보고 있어요.

추가 비용 없어요. 인테리어코치 안에 들어가 있는 기능이에요.
```

## 6. 카피 일관 키워드

스펙북 관련 카피에 일관되게 들어갈 단어 / 들어가지 말아야 할 단어.

**들어갈 단어 (권장)**
- 사장님 / 대표님
- 한 번 만들어 두고 우려쓴다
- 매번 새로 짜지 않는다
- 카탈로그 / 마감재 / 공종
- 카톡 / 엑셀 (현재 흐름)
- 링크 / 한 화면 / 누락
- 견적·발주·세금계산서가 그대로 흘러감

**들어가지 말아야 할 단어 (금지)**
- 혁신적인 / 압도적인 / 차별화된
- 올인원 솔루션 / 통합 플랫폼
- 디지털 전환 / 업무 혁신
- 페르소나 / 밸류 / 핏 / 퍼널 / SSOT
- 고객 경험 / CX
- 100% 자동화 / 0원 비용 / 완벽한
- 귀하 / 귀사 / 고객님
