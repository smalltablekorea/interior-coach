### 2-1. Awareness (인지) — "인테리어 업체 관리, 아직도 엑셀로 하세요?"

**목표:** 월 10,000 타겟 노출 달성

#### 채널별 전략

| 채널 | 전략 | 게시 빈도 | 예상 비용(월) | KPI |
|------|------|-----------|---------------|-----|
| **Threads (@realzeroto1)** | 창업 스토리텔링 + 인테리어 업계 인사이트 | 주 5회 | 0원 (오가닉) | 팔로워 증가율, 공유수 |
| **네이버 블로그** | SEO 최적화 키워드 콘텐츠 (장문) | 주 3회 | 0원 (오가닉) | 검색 유입수, 블로그 방문수 |
| **Instagram** | 비포/애프터 사례, 릴스, 카드뉴스 | 주 4회 | 월 50만원 (광고) | 도달수, 저장수 |
| **네이버 카페** | 인테리어 업자 커뮤니티 활동 + 정보 공유 | 주 2회 | 0원 (오가닉) | 카페 유입수, 댓글수 |
| **구글/네이버 검색광고** | "인테리어 견적 프로그램", "현장관리 앱" 키워드 | 상시 | 월 100만원 | CPC, CTR, 전환수 |

#### 콘텐츠 전략 (콘텐츠 3축)

```
[교육 콘텐츠 40%]              [공감 콘텐츠 30%]           [제품 콘텐츠 30%]
- 인테리어 세무 팁             - 업체 사장님 고충 공감      - 기능 데모 영상
- 견적서 작성 가이드           - "나만 이렇게 힘든가" 시리즈 - 고객 사례 인터뷰
- 현장관리 체크리스트          - 실수담/노하우 공유         - 업데이트 소식
- 마케팅 실전 가이드           - 업계 트렌드 분석           - 비교 분석 콘텐츠
```

**핵심 키워드 (SEO 타겟):**
- "인테리어 견적 프로그램" (월 검색량 추정 1,200)
- "인테리어 현장관리 앱" (월 검색량 추정 800)
- "인테리어 업체 관리" (월 검색량 추정 600)
- "인테리어 세무 관리" (월 검색량 추정 400)
- "인테리어 견적서 양식" (월 검색량 추정 3,000 — 견적코치 유입용)

### 2-2. Interest (관심) — "무료로 견적 분석해볼래요?"

**목표:** 방문자 40%를 견적코치 또는 랜딩페이지로 유도

#### 견적코치 무료 도구 (B2C -> B2B 브릿지)

```

      |                                   |
      v                                   v

      |                           쓸 수 있으면 좋겠다"
      v                                   |

 "업체용 SaaS 안내" CTA  <--------  직접 SaaS 랜딩 방문
      |                                   |
      v                                   v
 +-------------------------------------------------+
 |         SaaS 랜딩 페이지 (통합 유입)              |
 +-------------------------------------------------+
```

**견적코치 --> SaaS 전환 최적화:**
- 견적 분석 결과 페이지에 "업체 사장님이신가요? 이 기능을 고객에게 제공하세요" CTA 삽입

- 분석 데이터를 활용한 "시장 평균 단가 리포트" 리드 마그넷

**목표:** 가입자의 60%를 실제 체험 시작으로 전환

#### 14일 무료 체험 온보딩 플로우

```
 Day 0: 가입 직후
 ┌─────────────────────────────────────────────┐
 │  환영 모달                                   │
 │  "3가지만 설정하면 바로 시작할 수 있어요"      │
 │                                              │
 │  1. [ ] 업체 정보 입력 (상호명, 업종)         │
 │  2. [ ] 첫 번째 현장 등록                     │
 │  3. [ ] 첫 번째 견적서 작성                   │
 │                                              │
 │  예상 소요 시간: 5분                          │
 └─────────────────────────────────────────────┘
        |
 Day 1-3: 핵심 가치 경험
        |  - 인앱 코치마크로 주요 기능 안내
        |  - "현장 사진 올리기" 프롬프트
        |  - 견적서 PDF 출력 경험 유도
        |
 Day 4-7: 습관 형성
        |  - 매일 오전 9시 "오늘의 현장 일정" 푸시/이메일
        |  - 견적 요청 시뮬레이션 알림
        |  - 세무 기능 소개 팝업
        |
 Day 8-10: 가치 확인
        |  - "지금까지 절약한 시간" 리포트
        |  - "체험 기간 N일 남음" 알림
        |  - Pro 기능 맛보기 (1일 제한 해제)
        |

        |  - "체험 종료 D-3" 이메일

 Day 14: 최종 전환
        - 체험 만료 안내 + 원클릭 결제
```

### 2-5. Activation (활성화) — "이제 없으면 안 되는 도구로"

**목표:** 체험 사용자의 35%를 활성 사용자로 전환

#### 활성화 기준 정의

```
 ┌──────────────────────────────────────────────┐
 │            활성화 = 다음 3가지 모두 충족        │
 │                                               │
 │   [현장 1개 이상 등록]                         │
 │        +                                      │
 │   [계약/고객 1개 이상 등록]                     │
 │        +                                      │
 │   [견적서 1개 이상 작성 및 공유]                │
 │                                               │
 │   달성 시 → "축하합니다!" 모달 + 보상           │
 │   (첫 달 20% 할인 쿠폰 즉시 발급)              │
 └──────────────────────────────────────────────┘
```

#### 핵심 기능 사용 유도 (인앱 가이드)

```
 1단계: 코치마크 (첫 방문 시)
 ┌──────────────────────────────────────┐
 │  ┌──────────┐                        │
 │  │ 여기를    │  "현장을 등록하면       │
 │  │ 클릭!    │   일정/사진/메모를      │
 │  │   ↓      │   한 곳에서 관리할 수   │
 │  └──────────┘   있어요"              │
 │                                      │
 │  [다음]  [나중에]                     │
 └──────────────────────────────────────┘

 2단계: 빈 상태 페이지 (Empty State)
 ┌──────────────────────────────────────┐
 │                                      │
 │     현장이 아직 없어요               │
 │                                      │
 │     [+ 첫 번째 현장 등록하기]         │
 │                                      │
 │  "등록하면 일정 알림, 사진 관리,      │
 │   진행률 추적을 바로 시작할 수 있어요" │
 └──────────────────────────────────────┘

 3단계: 사용 맥락 추천 (Contextual)
 - 견적서 작성 완료 시 → "고객에게 카톡으로 바로 보내볼까요?"
 - 현장 등록 시 → "현장 사진도 함께 올려보세요"
 - 월말 접근 시 → "이번 달 세무 자료, 한 번에 정리해볼까요?"
```

### 2-6. Retention (유지) — "떠나지 않게, 계속 가치를 느끼게"

#### 이탈 조기 경보 시스템 (Health Score)

```
 고객 건강도 점수 (100점 만점)
```
