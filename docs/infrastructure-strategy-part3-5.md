## A. 규모별 세분화

### 1인 업체 (매출 1~3억)

**시장 규모:** 전체 인테리어 사업자 15만 중 약 60% = 9만 곳

| 항목 | 현실 |
|------|------|
| 동시 현장 | 1~2개. 현장 겹치면 품질 급락 |
| 디지털 도구 | 카카오톡 단체방 + 엑셀(또는 종이). 이것이 전부. 세무는 삼쩜삼 or 세무사 월 10~15만 |
| 월 IT 지출 | 0~5만 원. "무료 아니면 안 씀" |
| 결정권자 | 대표 = 영업 = 현장감독 = 경리. 1인이 전부 |
| 핵심 고충 | ① 견적서 만드는 데 반나절 ② 자재 단가 기억에 의존 ③ 수금 추적 불가 ④ 세금 신고 시즌 = 지옥 |

**"이거 없으면 안 돼"라고 느끼는 기능:**
1. **견적 자동 생성** — 반나절 걸리던 견적서를 10분에. 이것 하나로 가입 이유 성립
2. **수금/미수금 알림** — 현장 뛰면서 돈 받을 타이밍 놓치는 게 일상
3. **세무 시즌 데이터 추출** — 1년 치 영수증 모으는 고통을 없애면 연말에 "구독 유지" 결정

**가격 민감도: 극도로 높음.** 월 5만 원 이상이면 심리적 저항. "이거 공짜로 쓰다가 돈 내야 하면 그냥 카톡으로 돌아감."

**인테리어코치 진입 전략:**
- **무료 → Lite(4.9만)로 단계적 전환.** 무료에서 현장 1개 + 견적코치 월 3회를 열어주고, 2번째 현장이 생기면 Lite 전환 트리거
- 견적코치(B2C)로 먼저 만남 → "업체용 도구도 있습니다" 크로스셀
- 핵심 메시지: "카카오톡으로 현장 관리하면 매출의 15%를 놓친다" (미수금 + 자재 로스)

### 소규모 3~5인 (매출 3~10억) — 핵심 타겟

**시장 규모:** 전체의 약 25% = 3.75만 곳

| 항목 | 현실 |
|------|------|
| 조직 구조 | 대표(영업+경영) + 매니저 1~2명(현장 감독) + 반장 2~3명(외주) |
| 동시 현장 | 3~7개. "어떤 현장이 어디까지 진행됐는지" 대표 머릿속에만 존재 |
| 디지털 도구 | 카톡 + 엑셀 + 삼쩜삼/자비스 + 네이버 블로그(직접 or 대행) |
| 월 IT 지출 | 5~20만 원 (세무 10만 + 기타 5~10만) |
| 결정권자 | 대표 1인. 매니저 의견 참고하지만 결정은 대표 |

**현장 관리의 병목 지점:**
1. **정보 비대칭** — 대표는 사무실/영업 중, 현장 상황은 반장 카톡 보고에 의존. 반장이 답 안 하면 블랙아웃
2. **자재 발주 중복/누락** — "이 현장에 도배지 발주했나?" → 카톡 채팅 스크롤 100개 올림
3. **정산 지옥** — 현장별 수익률을 모름. "바빠서 돈은 벌었는데, 남는 게 없다"가 이 규모의 전형적 증상
4. **마케팅 단절** — 한 번 블로그 대행 맡기면 월 70~120만 원. ROI 측정 불가

**"이거 없으면 안 돼"라고 느끼는 기능:**
1. **멀티 현장 대시보드** — 7개 현장 상태를 한 화면에
2. **자재 발주/재고 추적** — 906개 자재 카탈로그 기반 자동 산출
3. **정산 리포트** — 현장별 수익률 자동 계산. "이 현장 마진 12%밖에 안 남았다"를 실시간으로
4. **반장/자재상 알림** — 대표가 직접 카톡 안 해도 자동 일정·발주 알림

**핵심 타겟인 이유:**
- IT 지출 15~30만 원 수용 가능 → Pro 29.9만 원 결제 가능 구간
- 성장 욕구 강함 — "10인까지 키우고 싶다"는 의지
- 현재 대안 없음 — 빌드온/하우빌드는 대형 건설용, 카톡+엑셀은 3현장 넘으면 붕괴
- 입소문 효과 — 반장이 3~4개 업체와 동시 작업 → 네트워크 전파 자연 발생
- **LTV 최대 구간** — 3~5인에서 10인으로 성장하면 Enterprise 업그레이드. 3~5년 고객 유지 기대

### 중형 10인+ (매출 10억+)

**시장 규모:** 전체의 약 10% = 1.5만 곳. 그러나 ARPU 최대.

| 항목 | 현실 |
|------|------|
| 조직 구조 | 대표 + 영업팀 + 설계팀 + 현장팀 + 경리/총무. 때로 다지점 |
| 동시 현장 | 10~30개. 현장별 매니저 배정 |
| 디지털 도구 | 자체 엑셀 시스템 or ERP(얼마에요, 세무사 법인 기장) + 마케팅 대행사 |
| 월 IT 지출 | 50~150만 원 |

**핵심 니즈:**
1. **팀 권한 관리** — 매니저별 현장 접근 분리, 반장은 자기 현장만
2. **경영 리포트** — 월별/분기별 매출·마진·현장 수익률 자동 집계
3. **다사업장 통합** — 본점+지점 데이터 통합 관리
4. **API 연동** — 기존 회계 시스템, ERP와 데이터 연결

**"이거 없으면 안 돼"라고 느끼는 기능:**
1. **역할 기반 접근 제어(RBAC)** — 반장/매니저/대표/경리 권한 분리
2. **통합 정산 대시보드** — 전 현장 합산 P&L
3. **커스텀 리포트 & 데이터 내보내기** — 세무사/회계사에게 넘길 데이터
4. **전용 지원** — 전담 CS, 온보딩 지원

**엔터프라이즈 플랜 가치:**
- 현재 분산 지출(세무 40만 + ERP 30만 + 마케팅 대행 100만 = 170만+)을 59.9만 원으로 통합
- 실질적으로 "월 110만 원 절감" 포지셔닝 가능
- 단, 이 세그먼트는 **기존 시스템 교체 저항이 강함**. "지금 돌아가는데 왜 바꿈?"
- 진입 전략: 정산 리포트 1개 기능만 먼저 도입 → 데이터 축적 → 전체 전환 유도 (트로이 목마)

## B. 업종별 세분화

### 주거 전문 (아파트/빌라 리모델링)

**시장 비중:** 전체 인테리어 시장 41조 중 약 65% = 26.6조

| 특성 | 내용 |
|------|------|
| 평균 공사 기간 | 3~6주 |
| 평균 공사 금액 | 2,000~8,000만 원 |
| 고객 소통 빈도 | 매우 높음. "타일 색 바꿔주세요" 하루 5~10회 변경 요청 |
| 마감재 선택 | 고객이 직접 선택. 결정 지연 = 공기 지연의 1위 원인 |
| 수주 경로 | 블로그 포트폴리오 → 상담 → 실측 → 견적 → 계약 |

**핵심 기능:**
- **견적코치** — 소비자가 먼저 분석한 견적을 가지고 옴 → 업체는 "우리도 견적코치로 보여드리겠습니다"로 신뢰 확보
- **고객 포털(토큰 기반)** — "공사 진행 상황 보여주세요" 카톡 질문 80% 감소. 시공 사진 자동 공유
- **마감재 선택 관리** — 자재 카탈로그 906개에서 고객이 선택 → 발주 자동 연동

**고객 포털의 가치가 가장 높은 세그먼트인 이유:**
- 주거 고객은 "내 집"이라 관심도가 극도로 높음
- 현재: 카톡으로 사진 보내고 → 읽씹 → 재전송 → "다른 각도로 찍어주세요" 반복
- 포털 도입 후: 자동 사진 업로드 + 진행률 표시 → 고객 만족도 상승 → 리뷰/추천 증가 → 신규 수주
- **"고객 포털 있는 업체 vs 없는 업체"가 곧 신뢰도 차이** = 차별화 무기

---

### 상업 전문 (카페/식당/오피스)

**시장 비중:** 약 25% = 10.25조

| 특성 | 내용 |
|------|------|
| 평균 공사 기간 | 2~4주 (임대료 나가므로 공기 단축 압박) |
| 평균 공사 금액 | 3,000만~2억 |
| 고객 소통 | B2B 관계. 임차인 or 프랜차이즈 본사. 의사결정 빠름 |
| 핵심 변수 | 임대차 계약 기간 내 완공 필수. 지연 = 임대료 손실 |
| 수주 경로 | 기존 관계(건물주/프랜차이즈) → 실측 → 견적 → 즉시 결정 |

**핵심 기능:**
- **일정관리(공기 관리)** — 공기 촉박. D-Day 카운트다운, 지연 현장 자동 경보
- **정산 리포트** — 상업 시공은 마진이 얇음(8~15%). 실시간 원가 추적이 생존 조건
- **반복 수주 관리(CRM)** — 프랜차이즈 본사와 관계 유지 → 20~50개 매장 연속 수주 가능

**규모와 반복성:**
- 1건 수주 = 1건 끝이 아님. 프랜차이즈 관계 유지하면 연간 10~50건 반복
- CRM에 "프랜차이즈 본사 담당자" 기록 + 자동 후속 연락 알림 = LTV 극대화
- 이 세그먼트에서 인테리어코치의 정산 리포트는 **"이 매장은 마진 12%, 저 매장은 8% — 왜?"**를 즉시 보여줌

### 겸업 (주거+상업)

**시장 비중:** 전체 업체의 약 50~60%. 가장 흔한 유형.

| 특성 | 내용 |
|------|------|
| 현장 다양성 | 아파트 3개 + 카페 1개 + 오피스 1개 동시 진행 |
| 관리 복잡도 | 현장마다 공정 다름, 자재 다름, 고객 유형 다름 |
| 핵심 고충 | "아파트 현장에서 쓰는 도배지와 카페에서 쓰는 도배지가 다른데, 발주를 섞었다" |

**핵심 기능:**
- **멀티 현장 관리** — 현장 유형별 필터링, 대시보드 커스텀
- **자재 카탈로그(906개)** — 주거용/상업용 자재 분류. 현장별 자동 매칭
- **통합 정산** — 주거 현장 마진 vs 상업 현장 마진 비교 → "우리 업체는 어디에 집중해야 하나" 의사결정

**"올인원"의 가치가 가장 큰 세그먼트:**
- 주거 전용 도구 + 상업 전용 도구를 따로 쓸 수 없음
- 인테리어코치의 범용성이 곧 USP
- **이 세그먼트가 인테리어코치의 코어 PMF(Product-Market Fit) 대상**

# Part 4: 가격 전략

---

## A. 현재 가격 분석

**현재 체계:** 무료 / 스타터 14.9만 / 프로 29.9만 / 엔터프라이즈 59.9만

### 강점

| 항목 | 분석 |
|------|------|
| 무료 Tier 존재 | 진입 장벽 제거. 인테리어 업체의 디지털 도입 거부감 완화 |
| Pro 29.9만 원 | Van Westendorp 최적 범위(15~30만) 상단. "비싸지만 납득 가능" 포지션 |
| 가격 앵커 | "블로그 대행 70만 + 세무사 35만 = 100만+ vs 29.9만" 비교 유효 |
| 4단계 구조 | 업그레이드 경로 명확 |

### 약점

| 항목 | 문제 | 근거 |
|------|------|------|
| **무료→14.9만 갭** | 무료에서 14.9만으로의 점프가 큼. 1인 업체 전환율 극저 예상 | 1인 업체 월 IT 지출 0~5만 원. 14.9만은 기존 IT 지출의 3배 |
| **스타터의 애매함** | 14.9만 원으로 뭘 해결할 수 있는지 불명확 | "Pro는 전부, 스타터는 일부"면 — 고객은 "일부"를 안 삼 |
| **엔터프라이즈 고정가** | 59.9만 원 고정가는 10인 업체와 50인 업체에 동일 적용 불가 | 대형 업체는 "커스텀 견적"을 기대. 고정가 = 저렴해 보여서 신뢰 하락 |

### 현장 1개 vs 3개 무료의 차이

| 무료 현장 | 장점 | 단점 | 판단 |
|----------|------|------|------|
| **1개** | 2번째 현장 = 유료 전환 자연스러움. 전환 퍼널 명확 | 1인 업체가 "현장 끝나면 탈퇴 → 재가입" 반복 가능 | **권장** |
| **3개** | 3~5인 업체도 무료 체험 가능. 더 깊은 경험 | 유료 전환 트리거 약함. 3현장이면 무료로 충분한 업체 다수 | 비권장 |

> **결론: 현장 1개 무료가 정답.** 단, "첫 현장 완공 후 데이터는 영구 보관"을 약속해서 데이터 이탈 방지.

### 데이터 기반 락인 가격 전략

**핵심 원리: 시간이 지날수록 이탈 비용이 증가하는 구조**

```
Month 1~3 (무료)
  ├── 현장 데이터 입력 중
  ├── 자재 발주 기록 축적
  ├── 작업자 연락처/평가 데이터 쌓임
  └── 아직 "다른 데로 옮겨도 됨"

Month 4~6 (전환 시점)
  ├── 정산 리포트: "이 데이터 3개월 치가 있어야 의미가 있습니다"
  ├── 세무 데이터: "부가세 신고에 이 데이터 그대로 쓸 수 있습니다"
  ├── 작업자 평가: "반장 40명의 평가 기록이 여기에만 있습니다"
  └── 이탈 비용 = 6개월 데이터 포기

Month 7~12 (락인 완성)
  ├── 1년 치 경영 데이터 = 세무사에게 넘길 원장
  ├── 고객 CRM: 과거 고객 연락처 + 시공 이력 = 재수주 자산
  └── "이걸 버리고 엑셀로 돌아갈 수 있나?" = 불가능
```

**"정산 리포트"를 유료 전용으로 잠그면?**
- 3개월 간 무료로 지출/수입 데이터를 입력하게 함
- 4개월 차에 "월간 정산 리포트가 준비되었습니다" 알림
- 클릭하면 → "정산 리포트는 Starter 이상에서 이용 가능합니다"
- **전환율 예상: 15~25%.** 3개월 데이터를 직접 넣은 사람은 "이거 보려고 넣었는데" 심리 작동

**"세무 회계"를 유료 전용으로 잠그면?**
- 부가세 신고 마감: 1월 25일, 7월 25일
- 12월~1월 / 6월~7월 = **유료 전환 최대 시즌**
- "인테리어코치에 입력한 데이터로 부가세 신고 자료를 자동 생성합니다" → Starter 이상
- **이 시즌에 집중 프로모션: "지금 Starter 가입하면 첫 달 50% 할인"**
- 전환율 예상: 신고 시즌 한정 30~40%

```
  ├── 업체 1: 인테리어코치 사용 중 → 알림으로 일정 받음
  ├── 업체 2: 카톡으로 일정 받음 → "업체 1은 알림 오던데..."
  ├── 업체 3: 전화로 일정 받음 → "요즘 다 앱으로 하던데요"
  └── 결과: 반장이 업체 2, 3에 "인테리어코치 쓰면 편합니다" 추천
```

**전파 시나리오:**
1. 업체 A가 인테리어코치에 반장 10명 등록
2. 반장 10명이 각각 3개 업체와 작업 = 30개 업체에 노출
3. 30개 업체 중 20%가 관심 = 6개 업체 가입
4. 6개 업체가 각각 반장 8명 등록 = 48명 추가 노출
5. **반복 → 네트워크 효과 시작**

**반장 유인 설계:**
- 반장 전용 앱(또는 알림 채널): 일정 확인, 완료 보고만 가능한 초경량 인터페이스
- 반장이 등록된 업체가 많을수록 반장의 일정 통합 관리 가능 → 반장 본인에게도 가치
- **"인테리어코치에서 일정 받는 반장" = 업계에서 신뢰도 높은 반장** 포지셔닝

### 자재상 협업 전략

**자재 카탈로그 906개를 통한 자재상 유입:**

| 단계 | 활동 | 자재상 가치 |
|------|------|-----------|
| Phase 1 | 자재 카탈로그에 자재상 정보 연동 (납품 가격, 배송 기간) | 신규 거래처 확보 채널 |
| Phase 2 | "이 자재를 취급하는 자재상 3곳 비교" 기능 | 자재상 간 경쟁 → 업체에 이익 |
| Phase 3 | 자재상 전용 대시보드: 발주 현황, 납품 일정, 미수금 관리 | 자재상도 인테리어코치 사용자로 전환 |

**자재상의 네트워크 파워:**
- 지역 자재상 1곳이 거래하는 인테리어 업체 = 20~50곳
- 자재상이 "인테리어코치로 발주해주세요, 관리가 편합니다" → 20~50곳 동시 유입
- **자재상 = 인테리어 업계의 허브 노드.** 이 노드를 장악하면 네트워크 급속 확산

### "반장이 3개 업체에서 인테리어코치 알림을 받으면" 시나리오

```
[업계 표준화 경로]

Year 1: 선도 업체 50곳 사용
  └── 반장 200명 노출, 자재상 20곳 노출

Year 2: 반장/자재상 추천으로 200곳 추가
  └── "인테리어코치 안 쓰는 업체는 반장 구하기 어렵다"

Year 3: 업계 표준
  └── "인테리어코치 = 인테리어 업체의 카카오톡"
  └── 안 쓰면 업무 진행 자체가 불편한 구조
```

이것이 "인프라"의 정의다. **선택이 아니라 전제 조건이 되는 것.**

## C. 견적코치(B2C) → 인테리어코치(B2B) 크로스셀

### 현재 견적코치 트래픽 활용 방안

```
[크로스셀 퍼널]

소비자: "인테리어 견적 3,500만 원 받았는데 적정한가?"
    │
    ▼
견적코치: "이 견적은 평균 대비 12% 높습니다. 특히 타일 공종이 비쌉니다."
    │
    ▼
소비자: "그럼 다른 업체 견적도 받아봐야겠네"
    │
    ├──── (B2C 경로) "근처 인테리어코치 파트너 업체 3곳의 견적을 비교해보세요"
    │         └── 업체 매칭 수수료 수익
    │
    └──── (B2B 경로) 소비자가 업체에 "견적코치에서 분석했는데요..."
              └── 업체: "우리도 견적코치 써야겠네" → 인테리어코치 가입
```

### 업체가 견적코치를 영업 도구로 활용하게 만드는 방법

| 전술 | 내용 | 기대 효과 |
|------|------|----------|
| **"견적코치 인증 업체" 배지** | 인테리어코치 Pro 구독 업체에 자동 부여 | 소비자 신뢰 → 업체 가입 유인 |
| **업체 전용 견적코치 링크** | "저희 업체 견적을 견적코치로 직접 검증해보세요" | 업체가 자발적으로 견적코치 홍보 |
| **견적 비교 리포트** | "A업체 견적 vs 시장 평균" 리포트를 업체가 고객에게 제공 | 투명성 = 신뢰 = 계약 전환율 상승 |
| **고객 포털 연동** | 견적코치 분석 결과 → 고객 포털에 자동 표시 | 견적~시공~완공 원스톱 경험 |

### 크로스셀 퍼널 전환율 목표

| 단계 | 전환율 목표 | 근거 |
|------|-----------|------|
| 견적코치 방문 → 분석 완료 | 40% | 무료 도구. 장벽 낮음 |
| 분석 완료 → "업체 연결" 클릭 | 15% | 견적이 비싸다고 나오면 대안 탐색 |
| 업체 연결 → 업체 노출 | 100% | 파트너 업체 DB 기반 자동 매칭 |
| 업체에게 "견적코치 분석 받은 고객 있다" 알림 → 업체 가입 | 10% | 실제 리드를 제공하므로 전환 동기 강함 |
| **총 견적코치 방문 → 업체 가입** | **0.6%** | 월 10,000 방문 기준 = 월 60곳 업체 가입 |

## D. 전략적 파트너십

### 1. 자재 유통사

| 파트너 | 협업 방안 | 인테리어코치 이익 | 파트너 이익 |
|--------|---------|-----------------|-----------|
| **한샘** | 한샘 자재를 카탈로그에 공식 등록. 한샘 거래 업체에 인테리어코치 추천 | 대형 유통망 접근 (한샘 거래 업체 수천 곳) | 디지털 발주 채널 확보. 업체 데이터 |
| **이케아 프로** | 이케아 비즈니스 회원 대상 공동 프로모션 | 상업 인테리어 세그먼트 접근 | 시공업체 대상 매출 확대 |
| **지역 자재상** | 자재 카탈로그에 지역 자재상 등록. 발주 연동 | 자재상 네트워크 = 업체 네트워크 | 온라인 발주 채널. 신규 거래처 |

**우선순위: 지역 자재상 > 한샘 > 이케아 프로**
- 지역 자재상은 의사결정이 빠르고, 각각 20~50개 업체와 거래
- 한샘은 협업 프로세스가 길지만, 성사 시 임팩트 최대
- 이케아 프로는 상업 세그먼트 한정

### 2. 인테리어 교육 기관

| 파트너 | 협업 방안 | 인테리어코치 이익 |
|--------|---------|-----------------|
| **인테리어 학원 (SPC, 그린 등)** | 수강생에게 인테리어코치 무료 계정 제공. 커리큘럼에 "인테리어코치로 현장 관리" 포함 | 졸업생 = 미래 고객. 브랜드 각인 시작점 |
| **직업훈련원 (폴리텍 등)** | 실습 도구로 인테리어코치 도입 | 공공기관 레퍼런스 확보. 신뢰도 상승 |
| **유튜브 인테리어 교육 채널** | 공동 콘텐츠 제작. "이 영상에서 사용한 견적 도구 = 인테리어코치" | 콘텐츠 마케팅 비용 절감. 타겟 정확 |

**핵심 논리:** 인테리어 업계에 진입하는 사람이 "첫 번째로 배우는 도구"가 되면, 평생 고객이 된다. 의대생이 첫 청진기 브랜드를 평생 쓰는 것과 같은 원리.

### 3. 부동산 중개소

| 협업 방안 | 메커니즘 |
|---------|---------|
| 매매/전세 계약 시 "인테리어 필요하시면" 연결 | 중개사 → 소비자에게 견적코치 추천 → 업체 연결 |
| 중개사 전용 견적코치 위젯 | 중개 앱/사이트에 견적코치 임베드 |
| 리모델링 수요 데이터 공유 | 어떤 단지에서 리모델링 수요가 높은지 → 업체에 인사이트 제공 |

**현실적 판단:** 부동산 중개소는 수수료 모델(건당 소개비)에만 반응한다. 인테리어코치 무료 제공만으로는 동기 부여 안 됨. 건당 2~5만 원 소개비 모델 필요 → 초기에는 ROI 불확실. **우선순위: 낮음. Year 2 이후 검토.**

### 4. 건축사 사무소

| 협업 방안 | 메커니즘 |
|---------|---------|
| 설계 완료 → 시공 업체 연결 | 건축사가 "시공은 인테리어코치 파트너 업체에서" 추천 |
| 설계 도면 → 인테리어코치 견적 자동 생성 | 건축사 → 업체 → 소비자 원스톱 |

**현실적 판단:** 건축사 사무소와 인테리어 시공업체는 기존 관계가 공고하다. 새 관계를 만들기보다, 기존 관계에 인테리어코치를 끼워넣는 방식이 현실적. **우선순위: 중간. Year 1 후반.**

> **이 문서의 결론:**
> 인테리어코치가 "업계 필수 인프라"가 되려면 기능의 완성도가 아니라 **네트워크의 밀도**가 관건이다.

> 그 임계점(Tipping Point)은 지역 내 30% 업체 도입 시점이다.
