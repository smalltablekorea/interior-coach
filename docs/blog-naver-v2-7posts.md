## 공통 템플릿

각 글 아웃라인은 다음 5단락 구조:

1. **상황 묘사 (후킹)** — 사장님이 자주 겪는 하루
2. **문제의 진짜 원인** — 왜 반복되는가
3. **기존 해결책의 한계** — 엑셀·카톡·대행 등
4. **인테리어코치 접근** — 어떤 모듈이 어떻게 대응하는지
5. **행동 제안 + 랜딩 링크**

분량 목표: 본문 1,500~2,200자. 이미지 3~5장 플레이스홀더 권장.

## 1) 인테리어 대표가 영수증 타이핑에 하루 두 시간 쓰는 이유

### SEO 키워드 (3)
- 인테리어 영수증 관리
- 인테리어 경비 정리 엑셀
- 현장 지출 OCR

### 제목 대안
- 주요: **"인테리어 대표가 영수증 타이핑에 하루 두 시간 쓰는 이유"** (30자)
- 보조: "현장 영수증 200장, 월말에 한번에 정리하는 법 바꾼 이야기"

### 아웃라인 (6)

- 현장 3개, 영수증 200장 — 한 달 기본 풍경
- 엑셀로 옮기는 데 실제로 걸리는 시간(밤 11시의 산수)
- 카톡 단톡방에 올린 사진이 금방 묻히는 구조
- 해외 SaaS의 QuickBooks+Dext 조합, 한국에선 비현실적
- 인테리어코치 "자재·발주 관리" 모듈 — 영수증 사진 한 장으로 자재명·수량·단가·공종이 자동 분류되는 흐름 (OCR 92~96%)
- 자동 분류된 값이 견적·손익·세금계산서까지 같은 숫자로 이어짐

### 랜딩 연결 한 줄

> 인테리어 영수증을 사진으로 찍으면 자동으로 정리되는 도구가 궁금하다면, 인테리어코치 14일 무료 체험에서 직접 써보실 수 있어요. → `interiorcoach.kr`

## 2) 현장 3개 돌리는 대표가 월말마다 통장 보며 놀라는 이유

### SEO 키워드 (3)
- 인테리어 현장 손익
- 다현장 원가 관리
- 인테리어 수익률

### 제목 대안
- 주요: **"현장 3개 돌리는 대표가 월말마다 통장 보며 놀라는 이유"** (29자)
- 보조: "세 현장 중 어느 현장에서 손해 나는지 몰랐던 사장님의 한 달"

### 아웃라인 (6)

- "매출은 괜찮은데 왜 통장이 허전하지" — 현장 간 손익이 섞여 보이지 않는 문제
- 엑셀 3개·카톡방 5개에 흩어진 지출 기록
- 일당 급여가 누구 현장에서 나간 건지 불분명한 상태
- 실시간 손익 비교가 중요한 현장 개수(3현장이 분기점)
- 인테리어코치 "현장 손익 관리" — 예산·실행·남은 금액을 다현장 나란히 비교하는 화면
- 손해 시그널(예산 90% 도달 알림) 자동 발송 → 손실 최소화

### 랜딩 연결

> 현장이 3개 넘는 순간부터는 손익을 "느낌"으로 관리할 수 없어요. 인테리어코치 다현장 손익 뷰를 14일 무료로 돌려보세요. → `interiorcoach.kr/pricing`

## 3) 세금계산서 한 번 놓치면 생기는 일

### SEO 키워드 (3)
- 인테리어 세금계산서
- 부가세 신고 자재
- 세금계산서 발행 기한

### 제목 대안
- 주요: **"세금계산서 한 번 놓치면 생기는 일"** (18자)
- 보조: "인테리어 업체 부가세 신고가 매번 꼬이는 진짜 이유"

### 아웃라인 (6)

- 1건 누락 = 수금 계획 꼬임 + 부가세 가산세
- 현장별로 매입·매출 세금계산서가 언제 발행됐는지 추적 어려움
- 세무사 카톡방이 결국 사장님 시간을 가져가는 구조
- 전통적 대응(다이어리·메신저 알림·엑셀 탭)이 실패하는 이유
- 인테리어코치 "세금계산서 관리" 모듈 — 매입·매출 자동 정리 + 발행 3일 전 알림
- 부가세 신고 직전 한꺼번에 정리하지 않아도 되는 워크플로우

### 랜딩 연결

> 세금계산서 발행 알림이 자동으로 나가는 도구를 쓰면 부가세 신고가 훨씬 수월해져요. 인테리어코치에서 직접 확인해보세요. → `interiorcoach.kr`

## 4) 하자 접수받고 잊어버리는 대표의 최후

### SEO 키워드 (3)
- 인테리어 하자 관리
- AS 접수 시스템
- 하자 사진 증빙

### 제목 대안
- 주요: **"하자 접수받고 잊어버리는 대표의 최후"** (21자)
- 보조: "AS 한 건 놓치면 브랜드 리뷰 3년 간다 — 인테리어 업체 하자 관리 실전"

### 아웃라인 (6)

- "누수인지 들뜸인지" 사진 받고 답만 하다 놓치는 전형
- 위치·사진·담당자가 분리되어 추적 불가능
- 엑셀 하자 대장의 현실 — 3주 뒤 업데이트가 멎는다
- 리뷰 1건의 장기 타격 (네이버·오늘의집·구글 검색에 남음)
- 인테리어코치 "하자관리" — 사진 AI가 공종·유형(누수·들뜸·깨짐)을 먼저 추정
- 도면 위 위치 마킹 + 처리 상태 추적 + 완료 증빙 자동 저장

### 랜딩 연결

> 하자 사진을 올리면 AI가 공종·유형을 먼저 추정합니다. 인테리어코치에서 도면 위 위치 마킹까지 직접 해보세요. → `interiorcoach.kr`

## 5) 반장님 구인이 매번 고민인 이유와 해결

### SEO 키워드 (3)
- 인테리어 반장 구인
- 인테리어 기사 인력
- 시공 협력업체 관리

### 제목 대안
- 주요: **"반장님 구인이 매번 고민인 이유와 해결"** (22자)
- 보조: "한 번 호흡 맞춰본 반장님을 다음 현장에서 또 부르는 법"

### 아웃라인 (6)

- 신규 현장마다 "지난번 그 반장님 번호 어디였지" 검색
- 평점·이력이 사장님 머릿속에만 있는 상태의 문제
- 건설워커·단톡방에 기대는 한계 (이력 누적 안 됨, 평점 없음)
- 크로스-업체 추천이 있으면 빨라지는 구인 속도
- 인테리어코치 "반장·기사 인력풀" — 함께 일한 평점·이력·현장 참여 기록 한 프로필에 누적
- 추천 기능: 동료 업체에 반장 추천 + 보상 받는 흐름

### 랜딩 연결

> 같이 일해본 반장님을 다음 현장에 바로 부를 수 있는 인력풀을 쓰고 싶다면 인테리어코치에서 확인해보세요. → `interiorcoach.kr`

## 6) 견적 한 번에 반나절 쓰는 대표를 위한 도구

### SEO 키워드 (3)
- 인테리어 견적 프로그램
- 자재 단가 DB
- 인테리어 견적서 자동

### 제목 대안
- 주요: **"견적 한 번에 반나절 쓰는 대표를 위한 도구"** (23자)
- 보조: "평수와 등급만 고르면 견적이 3분에 나오는 방법 (자재 DB 868건)"

### 아웃라인 (7)

- 고객 상담 → 현장 실측 → 견적서 만드는 데 반나절 가는 현실
- 공종별 단가를 매번 다른 곳에서 찾는 비효율
- 엑셀 양식을 고객마다 복사·수정하는 과정의 리스크(숫자 실수)
- 해외 Buildertrend·Houzz Pro의 "Selections" 개념 — 마감재 선택 워크플로우
- 인테리어코치 "견적코치 AI" — 868건 자재 DB + 3-Layer 가격 구조
- 평수·등급·공종 선택 → 3분 안에 견적 + 고객용 PDF 자동 생성
- 견적 → 계약 → 수금 → 세금계산서까지 같은 단가로 이어지는 라인

### 랜딩 연결

> 견적서 작성 시간을 실제로 재본 적 있으신가요? 인테리어코치 견적코치 AI로 다음 견적만 먼저 써보세요. → `interiorcoach.kr`

## 7) 인테리어 업체 사장님의 4월이 항상 위태로운 이유

### SEO 키워드 (3)
- 인테리어 현금 흐름
- 인테리어 월간 리포트
- 부가세 1분기 신고

### 제목 대안
- 주요: **"인테리어 업체 사장님의 4월이 항상 위태로운 이유"** (26자)
- 보조: "1분기 부가세·3월 잔금·4월 자재값이 겹치는 한 달, 무엇이 흔들리나"

### 아웃라인 (7)

- 4월에만 몰리는 3가지 — 1분기 부가세 신고 / 전 현장 잔금 수금 / 4월 착공 자재 발주
- 각각 엑셀·카톡·통장 조회로 따로 관리하다 보면 겹쳐 보이는 현금 위기의 실체
- "받을 돈은 많은데 잔고가 비는" 역설의 구조
- 월간 리포트가 있으면 바뀌는 점(현장별 손익 + 수금 현황 + 세금 스케줄이 한 장)
- 인테리어코치 "월간 리포트" — 매월 1일 자동 이메일 발송, 세무사 제출용 한 장
- "수금관리" + "세금계산서 관리" + "현장 손익 관리" 3모듈 결합 효과
- 4월을 관리 가능한 달로 만드는 실전 루틴 (월말 마감 3단계)

### 랜딩 연결

> 현금 흐름이 겹치는 달일수록 자동 리포트가 힘이 됩니다. 인테리어코치 월간 리포트를 14일 무료로 확인해보세요. → `interiorcoach.kr`
