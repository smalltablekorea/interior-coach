# 인테리어코치 — 투자 IR 원페이저

> 스몰테이블 | 인테리어 업체를 위한 올인원 SaaS

---

## Problem

국내 인테리어 시장 **41조 원**. 업체의 90%가 5인 미만 영세 사업자.

이들은 현장관리, 마케팅, 세무, 견적을 **4개의 서비스에 분산**해서 쓰고 있다:
- 블로그 대행: 월 50~120만 원
- 세무사: 월 15~40만 원
- 현장관리앱: 월 3~10만 원
- SNS 대행: 월 40~150만 원
- **합계: 월 100~300만 원+**

인테리어 업체를 위한 통합 SaaS는 **국내에 단 하나도 없다.**

---

## Solution

**인테리어코치** = 인테리어 업체의 모든 업무를 하나로.

| 기능 | 대체 대상 | 월 비용 절감 |
|------|----------|-------------|
| 현장관리 (공정, 사진, 일정) | 빌드온, 시공왕 (3~10만) | 3~10만 원 |
| 견적코치 AI (12공종 분석) | 집첵 (B2C) | 신규 수익원 |
| 마케팅 자동화 (블로그·인스타·스레드·유튜브) | 대행사 (70~300만) | 70~300만 원 |
| 세무/회계 (OCR, 부가세, AI 상담) | 삼쩜삼, 세무사 (10~40만) | 10~40만 원 |

**인테리어코치 Pro = 월 29.9만 원 (연간 23.9만)에 전부 포함.**

## Business Model

**3중 수익 구조:**

```
① SaaS 구독 (B2B)          ② 견적코치 건당 (B2C)       ③ 매칭 수수료
   Free → Starter ₩119K       프로 분석 ₩39,900/건        소비자 → 업체 연결
   → Pro ₩239K                                            시 건당 수수료
   → Enterprise ₩479K
```

**양면 플랫폼 플라이휠:**
```
견적코치(소비자) → 고객 유입 → 업체 연결 → 인테리어코치(업체 SaaS) → 데이터 축적 → 더 정확한 분석 → 더 많은 소비자
```

## 경쟁 우위 (Moat)

1. **데이터 모트**: 자체 시공사 운영 → 실제 원가 DB (경쟁사 모방 불가)
2. **네트워크 효과**: 견적코치 소비자 ↔ SaaS 업체 양면 네트워크
3. **올인원 통합**: 4개 영역을 하나로 묶은 국내 유일 서비스
4. **전환 비용**: 현장/고객/견적/세무 데이터 락인
