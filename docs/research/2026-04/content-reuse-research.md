### 핵심: 반장이 찍는 사진 = 콘텐츠

반장이 인테리어코치에 업로드하는 시공 사진이 자동으로:
1. 고객 포털에 노출 (고객 경험)
2. SNS 콘텐츠 소재로 활용 (마케팅)
3. 케이스 스터디 증거 (영업)

**다솜 추가 업무: 없음.** 반장이 원래 하던 사진 업로드가 3중으로 활용됨.

## 5. 결론: 재활용 맵

### 기본 권고안: "3단계 재활용 시스템"

**1단계 (즉시, 다솜 0분):**
- @realzeroto1 기존 포스트에서 인용문 10개 발췌 → 랜딩 카피에 삽입
- 맴맴곳곳 기존 영상에서 10초 클립 → 랜딩 히어로 배경 (해원이 편집)
- 기존 블로그 시공사례 하단에 인테리어코치 CTA 배너 추가 (은영)

**2단계 (4/16~5/4, 잠실르엘 자동 수집):**
- 반장이 인테리어코치에 업로드하는 사진 = 콘텐츠 소재 자동 수집
- 해원이 촬영하는 영상 → YouTube + 랜딩 동시 활용
- 공정별 스크린샷 → 기능 데모 섹션 자동 갱신

**3단계 (5/4 이후, 케이스 스터디 완성):**
- Before/After 비교 → 랜딩 메인 비주얼
- 18일 공정 타임라인 → "이렇게 관리했습니다" 스토리
- 고객 포털 실사용 스크린샷 → "고객이 이렇게 확인합니다" 섹션

### 대안: "반장 사진 없이도 돌리는 최소 버전"

잠실르엘 반장이 사진을 안 올려도:
- @realzeroto1 기존 포스트 + 맴맴곳곳 기존 영상만으로 랜딩 소셜프루프 구성 가능
- 작업지시서(이미 있음)를 셀프 온보딩 샘플 데이터로 활용
- 최소 랜딩: 히어로 카피 + 인용문 5개 + CTA만으로 1페이지 완성
