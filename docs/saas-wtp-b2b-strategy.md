### 1-2. 인테리어 업체 IT 지출 현황

### 1-4. Van Westendorp 가격 민감도 분석 (예상 결과)

5~10인 인테리어 업체 대상 예상 응답 분포:

| 질문 | 예상 응답 중위값 |
|------|---------------|
| "이 가격이면 너무 싸서 품질이 의심된다" (Too Cheap) | 5만 원/월 |
| "이 가격이면 싸다고 느낀다" (Cheap/Bargain) | 15만 원/월 |
| "이 가격이면 비싸지만 구매를 고려한다" (Expensive) | 30만 원/월 |
| "이 가격이면 너무 비싸서 구매하지 않겠다" (Too Expensive) | 50만 원/월 |

**최적 가격대 (OPP)**: **15~30만 원/월**
→ 현재 Pro 가격 23.9만 원은 최적 범위 내

### 1-5. 권장 가격 체계

| 플랜 | 가격 | 타겟 | 핵심 기능 |
|------|------|------|----------|
| **Free** | 무료 | 1인 사업자, 체험 | 현장 1개, 견적코치(월 3회), 기본 대시보드 |
| **Starter** | 9.9만 원/월 | 2~3인 소형 | 현장 3개, 견적코치(무제한), 업무일지, 기본 수금관리 |
| **Pro** | 23.9만 원/월 | 4~10인 중소 | 현장 무제한, 전 기능, AI 마케팅, 세무 가이드, 하자관리 |
| **Enterprise** | 커스텀 (50만+) | 10인+ 대형 | 다사업장, 전용 지원, API, 커스텀 리포트 |

**가격 앵커링 전략:**
- "블로그 대행 70만 + 세무사 35만 + 현장앱 15만 = 120만+/월" → Pro 23.9만 원
- 연간 결제 시 20% 할인 → 월 19.1만 원 (연 229만 원)

## 2. B2B 진입 전략

### 2-1. 내부 도구 → 외부 SaaS 전환 로드맵

| 단계 | 기간 | 활동 | 목표 |
|------|------|------|------|
| **Phase 0: 자체 사용** | M1~M3 | 스몰테이블에서 실제 시공 현장에 사용 | 제품-시장 적합성 검증, 버그 해결 |
| **Phase 1: 클로즈드 베타** | M4~M6 | 인천 송도 인근 5~10개 업체 무료 제공 | 피드백 수집, 기능 보완, 초기 사례 확보 |
| **Phase 2: 오픈 베타** | M7~M9 | 수도권 확대, Freemium 모델 | 100개 업체 가입, 유료 전환 테스트 |
| **Phase 3: 정식 런칭** | M10~M12 | 전국 확대, 마케팅 본격화 | 500개 업체, 유료 50개, MRR 1,000만 |

### 2-2. 유통 채널

| 채널 | 접근 방법 | 예상 CAC | 전환율 |
|------|----------|---------|--------|
| **기존 고객 (인천 송도)** | 스몰테이블 네트워크 | 0원 | 높음 |
| **인테리어 협회/조합** | 대한인테리어산업협회, 지역 인협 교육 세미나 | 5~10만/업체 | 중간 |
| **네이버 카페** | 인테리어 업자 커뮤니티 (인테리어114 등) 후기 마케팅 | 3~5만/업체 | 중간 |
| **인스타그램 광고** | 인테리어 업체 대표 타겟 광고 | 5~15만/업체 | 낮음~중간 |
| **자재 유통사 제휴** | 자재 납품처 → 고객사 소개 | 2~5만/업체 | 높음 |
| **교육기관** | 인테리어 학원/직업훈련원 제휴 | 1~3만/업체 | 낮음 (장기) |
| **Threads/블로그 오가닉** | 인테리어 업체 경영 노하우 콘텐츠 | 0~1만/업체 | 낮음 (장기) |

### 2-3. 파트너십 기회

| 파트너 유형 | 구체적 대상 | 협업 모델 | 가치 교환 |
|------------|-----------|----------|----------|
| **자재 유통사** | 한샘, 에넥스, 현대리바트 | 견적코치에 자재 카탈로그 연동 | 유통사: 판매채널 / 인코: 데이터+고객 |
| **세무법인** | 인테리어 업종 전문 세무사 | 세무 기능 내 세무사 매칭 | 세무사: 고객유입 / 인코: 전문성 |
| **부동산 중개** | 입주 예정 아파트 인근 공인중개사 | 입주민 → 인테리어 업체 → 인코 | 중개사: 커미션 / 인코: 리드 |
| **보험사** | 시공 배상책임보험 | 인코 가입 업체 단체 할인 | 보험사: 판매채널 / 인코: 부가가치 |
| **금융사** | 카카오뱅크, 하나캐피탈 | 인테리어 대출 연계 | 금융사: 대출 판매 / 인코: 고객편의 |

### 2-4. Freemium → Premium 전환 퍼널

```
[인지] 블로그/SNS/광고/입소문
  ↓ (방문)
[가입] Free 플랜 가입 (이메일/소셜)
  ↓ (7일 내 첫 견적 작성)
[활성화] 견적코치 3회 사용, 현장 1개 등록
  ↓ (14일 무료 Pro 체험)
[체험] Pro 기능 맛보기 (AI 마케팅, 다현장, 수금관리)
  ↓ (체험 종료 3일 전 알림)
[전환] Pro 유료 전환 (23.9만/월)
  ↓ (3개월 사용)
[정착] 데이터 축적 → 이탈 비용 증가
  ↓ (6개월)
[확장] Enterprise 또는 추가 사용자 구매
```

**핵심 전환 트리거:**
1. 견적코치 무료 3회 소진 → "더 분석하려면 Starter로"
2. 현장 3개 초과 → "다현장 관리는 Pro에서"
3. 14일 Pro 체험 후 기능 제한 → "지금 전환하면 첫 달 50% 할인"

## 3. 시장 규모 추정

### 3-1. TAM (Total Addressable Market)

| 항목 | 수치 | 근거 |
|------|------|------|
| 국내 인테리어/리모델링 시장 | **41조 원** (2025) | 한국건설산업연구원 |
| 연평균 성장률 | 5~7% | 1인 가구 증가, 노후 주택 리모델링 수요 |
| 2026년 추정 | **43~44조 원** | |

### 3-2. SAM (Serviceable Addressable Market)

| 항목 | 수치 | 산출 |
|------|------|------|
| 전국 인테리어 사업체 수 | ~120,000개 | 통계청 사업체조사 (실내건축 및 인테리어) |
| SaaS 도입 가능 업체 (3인+) | ~30,000개 | 전체의 ~25% |
| 업체당 연 SaaS 지출 가능액 | ~120만 원 | 월 10만 원 평균 |
| **SAM** | **~3,600억 원** | 30,000 x 120만 |

### 3-3. SOM (Serviceable Obtainable Market)

| 기간 | 목표 업체 수 | ARPU (월) | MRR | ARR |
|------|-----------|----------|-----|-----|
| 1년차 | 200개 | 15만 원 | 3,000만 | 3.6억 |
| 2년차 | 1,000개 | 18만 원 | 1.8억 | 21.6억 |
| 3년차 | 3,000개 | 20만 원 | 6억 | 72억 |

**시장 점유율**: 3년차 기준 SAM의 ~2% (3,000/30,000 x 200만/120만)

### 3-4. Unit Economics

| 지표 | 수치 | 산출 근거 |
|------|------|----------|
| **ARPU** (월 평균 객단가) | 15~20만 원 | Free 포함 블렌디드 |
| **CAC** (고객 획득 비용) | 10~30만 원 | 채널별 가중 평균 |
| **LTV** (고객 생애 가치) | 360~720만 원 | ARPU 20만 x 24개월 평균 유지 x 75% 마진 |
| **LTV/CAC 비율** | **12~36:1** | 건강한 수준 (3:1 이상 권장) |
| **CAC Payback** | **1~2개월** | CAC 20만 / 월 마진 15만 |
| **월 Churn Rate** | 3~5% (추정) | SaaS 평균, 데이터 모트로 개선 가능 |
| **Gross Margin** | 70~80% | 클라우드 비용, AI API 비용 차감 |

### 3-5. 손익분기점 (BEP)

| 항목 | 월 비용 (추정) |
|------|-------------|
| 클라우드 인프라 (Vercel, Neon) | 30~50만 |
| AI API 비용 (OpenAI, Claude) | 50~100만 |
| 인건비 (개발자 2인, 마케팅 1인) | 1,200만 |
| 마케팅 비용 | 200~500만 |
| 기타 (법무, 회계, 사무) | 100~200만 |
| **총 월 고정비** | **~2,000만** |

**BEP 유료 고객 수**: 2,000만 / 20만(ARPU) = **~100개 유료 업체**
**BEP 도달 예상**: 런칭 후 **8~12개월**

## 4. B2B 1페이지 요약

---

### 인테리어코치 — B2B SaaS 사업 요약

**한 줄 요약**: 한국 인테리어 업체를 위한 유일한 올인원 경영관리 SaaS

#### 시장 기회
- 인테리어 시장 41조 원, SaaS 도입 가능 업체 3만 개
- 현재 올인원 솔루션 부재 → 파편화된 도구 사용 (평균 4~5개)
- 해외 검증 모델 (Houzz Pro, Buildertrend) → 한국 로컬라이제이션

#### 제품
- **견적코치** (B2C 유입): AI 견적 분석으로 소비자 유입 → 업체 연결
- **인테리어코치** (B2B SaaS): 현장관리 + 하자관리 + 마케팅 + 세무 + CRM
- 9가지 핵심 기능을 월 23.9만 원에 통합 (경쟁사 대비 1/5 비용)

#### 비즈니스 모델
- Freemium (무료 → Starter 9.9만 → Pro 23.9만 → Enterprise 커스텀)
- LTV/CAC 12~36:1, CAC Payback 1~2개월
- BEP: 유료 100개 업체 (런칭 후 8~12개월)

#### 차별화
- **AI 네이티브**: 견적 분석, 마케팅 콘텐츠 자동 생성, 하자 분류
- **B2C+B2B 양면**: 견적코치로 소비자 유입 → 업체 고객으로 전환
- **현업 기반**: 실제 인테리어 회사(스몰테이블)에서 검증 후 출시

#### Traction (계획)
| 마일스톤 | 시기 | 목표 |
|---------|------|------|
| MVP 내부 사용 | M1~M3 | 자체 현장 5개 운영 |
| 클로즈드 베타 | M4~M6 | 10개 업체, NPS 50+ |
| 오픈 베타 | M7~M9 | 100개 가입, 20개 유료 |
| 정식 런칭 | M10~M12 | 500개 가입, 100개 유료 |
| 성장 가속 | Y2 | 3,000개 가입, 1,000개 유료, ARR 21.6억 |

#### 팀
- **대표** (인테리어 디자이너 출신) — 도메인 전문성
- **개발팀** — Next.js, AI 기반 풀스택
- 총 6인 구성 (스몰테이블디자인그룹, 인천 송도)

#### 시장 진입 전략
1. 자체 사용으로 제품 검증 → 2. 인천 지역 업체 확대 → 3. 수도권 → 4. 전국
- 핵심 채널: 인테리어 협회/조합, 네이버 카페, 자재 유통사 제휴
- 레퍼런스 고객 전략: 스몰테이블 = 첫 성공 사례

#### 리스크 & 대응
| 리스크 | 대응 |
|--------|------|
| 오늘의집 B2B 확장 | 자체 마케팅 역량 + AI 차별화로 선점 |
| 낮은 IT 리터러시 | 극도로 심플한 UX + 온보딩 + 고객 지원 |
| 가격 저항 | 무료 플랜 + 14일 체험 + ROI 명확히 증명 |
| Churn | 데이터 모트 구축 (견적/고객/현장 데이터 축적) |
