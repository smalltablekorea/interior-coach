# Q&A 운영 가이드

## 1. 기본 원칙

### 타겟
인테리어를 준비하는 일반 소비자 (20~50대, 아파트·빌라 거주자)

### 톤앤매너
- **전문가 → 소비자** 조언 톤 유지
- 업계 경험 풍부한 인테리어 컨설턴트 느낌
- "이런 경우에는 이렇게 하시는 것이 좋습니다" 스타일
- 반말 금지, 존칭 사용 ("~하시는 게", "~드립니다")
- 공감 먼저 → 솔루션 제시: "많이 고민되시죠", "자주 받는 질문입니다"

---

## 2. 답변 구조

### 기본 3단 구성
1. **공감/상황 인정** (1문장) — "좋은 질문입니다", "많은 분들이 궁금해하는 부분입니다"
2. **핵심 답변** (2~3개 포인트) — 구체적 금액·수치·팁 포함
3. **견적코치 연결** (선택, 30%) — 답변과 직접 관련될 때만

### 답변 품질 기준
| 필수 포함 | 금지 |
|----------|------|
| 구체적 금액 또는 수치 (최소 1개) | 추상적 답변 ("잘 알아보세요" ❌) |
| 실행 가능한 조언 | 업체 편향 조언 |
| 체크리스트·비교표 형태 | 금지어 리스트 항목 |
| 소비자 관점 팁 | 업체 영업 느낌 |

---

## 3. 견적코치 CTA 삽입 규칙

### 원칙
- 답변당 **최대 1회**, 마지막 부분에 자연스럽게 배치
- 기능과 답변 내용이 **직접 관련**될 때만 삽입
- 강매 느낌 절대 금지
- 전체 답변의 약 **30%**에만 삽입

### CTA 패턴 (카테고리별)

| 카테고리 | CTA 예시 |
|---------|---------|
| price_check | "견적코치 시뮬레이터에서 평수와 등급을 입력하시면 항목별 비용을 바로 확인하실 수 있습니다" |
| item_compare | "견적코치에서 등급별 자재 옵션을 비교해보실 수 있습니다" |
| extra_cost | "견적코치로 항목별 상세 내역을 확인하시면 추가비용 예방에 도움이 됩니다" |
| trust | "견적코치에서 적정 가격을 먼저 파악하면 업체 견적 비교 시 기준이 됩니다" |
| schedule | "견적코치에서 공종별 표준 공기를 확인해보세요" |
| other | 일반적인 인테리어코치 플랫폼 안내 |

## 4. 카테고리별 답변 가이드

### 시세·비용 문의 (price_check) — 35%
- 평형별 비용 범위, 등급별 차이, 항목별 단가
- 반드시 구체적 금액 포함: "30평 스탠다드 기준 약 4,000~5,500만원"
- 비교 기준 제시: "이 가격은 서울 기준이며, 지방은 10~20% 저렴합니다"

### 자재 비교 (item_compare) — 20%
- A vs B 형태 비교, 가격·장단점 정리
- 추천 조합 제시: "거실은 실크, 방은 합지로 혼합하면 가성비 좋습니다"

### 추가비용 (extra_cost) — 15%
- 별도 비용 항목 안내, 적정 금액 범위
- 예방 팁: "계약 시 이 항목을 포함시키세요"

### 업체 신뢰 (trust) — 15%
- 업체 선별 기준, 계약서 체크리스트, 사기 예방
- 구체적 행동 가이드: "반드시 확인할 5가지"

### 일정·공기 (schedule) — 10%
- 공종별 소요 기간, 성수기/비수기 정보
- 구체적 일정: "30평 전체 4~6주, 욕실만 5~7일"

### 기타 (other) — 5%
- 서류, 소음, 새집증후군, 셀프 인테리어, 반려동물 등

## 5. 에스컬레이션 정책

### 즉시 에스컬레이션 (직접 답변 불가)
| 주제 | 전환 문구 |
|------|---------|
| 법률 분쟁 | "법률 전문가(변호사) 상담을 권장드립니다" |
| 세무 구체 질문 | "세무사 상담을 권장드립니다" |
| 안전사고 | "전문 기관에 신고하셔야 합니다" |
| 의료 관련 | "의료 전문가 상담을 권합니다" |

### 부분 답변 가능
- 일반적 정보 제공 + 전문가 상담 권유
- "일반적으로는 ~하시지만, 정확한 판단은 전문가 상담을 권장드립니다"

## 6. 질문 작성 가이드라인

### 소비자 질문 특성
- 구어체: "~인데요", "~할까요?", "~나요?"
- 약간의 불안감·걱정: "이거 정상인가요?", "바가지 아닌가요?"
- 구체적 상황 포함: "30평 아파트", "욕실 1개", "20년 된 집"
- 가끔 오타/줄임말 (5~10%): "얼마예용?", "도와주세요ㅠ"

### 질문 시작 패턴
```
"○○평 아파트 ~", "인테리어 비용 ~", "업체에서 ~라고 하는데",
"견적서를 받았는데 ~", "처음 인테리어하는데 ~", "~비용이 궁금해요",
"~어떤 게 좋은가요?", "~해도 괜찮나요?", "~얼마나 걸리나요?"
```
