# Q&A 금지어 리스트

## 1. 경쟁사 직접 비방 금지

| 금지 | 허용 |
|------|------|
| "집닥은 비싸다" 등 직접 비방 | 일반적 비교: "플랫폼 경유 시 수수료 포함" |
| "오늘의집 믿지 마세요" | "플랫폼 리뷰도 참고하되 직접 확인이 중요합니다" |
| 특정 업체명 비방 | "일부 업체의 경우~" |

> 경쟁 플랫폼(오늘의집, 집닥, 숨고 등)을 언급할 수 있지만, **비방·폄하는 금지**
> 객관적 비교나 장단점 설명은 허용

---

## 2. 불법/비윤리 관련 금지

- 무허가 증축·불법 건축 조장
- 무자격 시공 권유 (전기·설비·방수 등 자격 필요 공종)
- 탈세·세금 회피 방법 안내
- 관리사무소 신고 없이 공사하는 방법
- 불법 폐기물 투기 방법
- 산업안전 규정 우회 방법

---

## 3. 부정 표현 전환

| 금지어 | 대체 표현 |
|-------|----------|
| 사기 업체 | 신뢰도가 낮은 업체, 주의가 필요한 업체 |
| 먹튀 | 계약 불이행, 연락 두절 |
| 날림공사 | 시공 품질 부족, 마감 미흡 |
| 바가지 | 시세 대비 과다 청구, 적정 가격 초과 |
| 호구 | 정보 부족으로 불리한 계약 |

> 소비자 질문에서 "사기", "먹튀" 등 사용은 자연스러우므로 질문에는 허용
> 답변에서는 반드시 대체 표현 사용

---

## 4. 개인정보 관련

| 금지 | 대체 |
|------|------|
| 특정 업체 실명 비방 | "일부 업체", "주의가 필요한 경우" |
| 특정 아파트 단지명 + 부정적 내용 | "○○구 아파트", "수도권 신축" |
| 개인 연락처·주소 | 삭제 또는 마스킹 |

---

## 5. 에스컬레이션 전환 표현

답변 불가 주제에 대한 표준 전환 문구:

| 주제 | 표준 전환 문구 |
|------|-------------|
| 법률/계약 분쟁 | "정확한 법적 판단은 법률 전문가(변호사) 상담을 권장드립니다" |
| 세무/세금 | "세무 관련 구체적 사항은 세무사 상담을 권장드립니다" |
| 안전사고 | "안전 관련 문제는 전문 기관에 신고·상담하시길 권합니다" |
| 보험/보상 | "보험 관련은 해당 보험사 또는 전문가에게 문의하시길 권합니다" |
| 의료(새집증후군 심각) | "건강 관련 증상은 의료 전문가 상담을 권합니다" |

### 전환 패턴
```
❌ "소송하세요"
✅ "소비자원에 분쟁조정을 신청하거나 법률 전문가 상담을 권장드립니다"

❌ "세금 이렇게 하면 됩니다"
✅ "일반적으로는 ~하시지만, 정확한 세무 처리는 세무사 상담을 권장드립니다"

❌ "그 업체 사기업체입니다"
✅ "계약 전 반드시 사업자등록 확인, 계약서 작성, 공정별 대금 분할을 지키세요"
```

---

## 6. 적용 범위

- **질문(title, content)**: 소비자 구어체이므로 "사기", "먹튀" 등 자연스러운 표현 허용
- **답변(answer)**: 모든 금지어 대체 표현 필수 적용, 에스컬레이션 전환 필수
- **카테고리/태그**: 금지어 미포함

---

## 7. 검수 프로세스

1. 생성된 시드 데이터에서 금지어 자동 필터링 (`grep` 활용)
2. 발견 시 대체 표현으로 즉시 수정
3. 에스컬레이션 주제 답변은 전환 문구 포함 여부 확인
