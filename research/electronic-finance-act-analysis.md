# 전자금융거래법 분석: 견적코치 "1회 분석권"의 선불전자지급수단 해당 여부

> **작성 목적**: 토스페이먼츠 가맹점 심사 과정에서 "크레딧"이라는 용어가 선불전자지급수단 의심을 받은 것에 대응하여, 견적코치의 1회 분석권이 전자금융거래법상 규제 대상이 아님을 법적으로 입증하기 위한 분석 문서
>
> **결론 요약**: 견적코치 분석권은 선불전자지급수단이 아니며, 전자금융업 등록 의무가 없다.

## 1. 전자금융거래법 제2조 — 선불전자지급수단의 법적 정의 {#1}

### 조문 원문

**전자금융거래법 제2조 제14호** (2023년 9월 14일 개정, 법률 제19734호):

> "선불전자지급수단"이라 함은 **이전 가능한 금전적 가치**가 전자적 방법으로 저장되어 발행된 증표(전자적 방법으로 변환되어 저장된 증표를 포함한다) 또는 그 증표에 관한 정보로서 **발행인(대통령령으로 정하는 특수관계인을 포함한다) 외의 제3자**로부터 재화 또는 용역을 구입하고 그 대가를 지급하는데 **사용되는 것**을 말한다. 다만, 전자화폐를 제외한다.

### 선불전자지급수단 성립의 3대 요건

학설과 금융당국의 유권해석을 종합하면, 어떤 수단이 선불전자지급수단이 되려면 다음 **3가지 요건을 모두** 충족해야 한다.

| 요건 | 내용 | 핵심 포인트 |
|------|------|-------------|
| **① 이전가능한 금전적 가치** | 전자적 방법으로 저장된 증표 또는 정보가 금전적 가치를 가지며, 타인에게 이전(양도)할 수 있어야 한다 | 단순 서비스 접근 권한은 해당 없음 |
| **② 전자적 저장·발행** | 금전적 가치가 전자적 방법으로 저장·발행되어야 한다 | 실물 상품권도 전자화되면 포함 |
| **③ 제3자 가맹점 사용 가능성** | 발행인 외의 제3자가 운영하는 가맹점에서 재화·용역 구매 대가로 사용 가능해야 한다 | **이것이 핵심 — 자사 서비스 전용이면 해당 없음** |

### 가장 중요한 요건: 제3자성(第三者性)

법조문의 핵심은 **"발행인 외의 제3자"** 라는 표현이다. 이는 포인트·크레딧이 아무리 금전적 가치를 가지더라도, 오직 **발행인 자신의 서비스에서만** 사용될 수 있다면 선불전자지급수단에 해당하지 않는다는 의미이다.

즉, 다음 두 구조는 본질적으로 다르다:

```
[선불전자지급수단 O]
이용자 → 카카오에 현금 충전 → 카카오페이 잔액 → A마트 결제 + B식당 결제 + C쇼핑몰 결제
                                               ↑
                              "발행인(카카오) 외 제3자 가맹점" ← 핵심 요건 충족

[선불전자지급수단 X]
이용자 → 견적코치에 결제 → 분석권 1회 취득 → 견적코치 서비스 내에서만 사용
                                               ↑
                              "발행인(견적코치) 자체 서비스만" ← 핵심 요건 미충족
```

## 2. 전자금융거래법 제28조 — 등록 의무와 면제 기준 {#2}

### 전자금융업 등록 의무의 원칙

**전자금융거래법 제28조 제1항**은 선불전자지급수단의 발행 및 관리업을 영위하려는 자는 금융위원회에 등록해야 한다고 규정한다.

등록 요건:
- 최소 자본금 요건
- 전산설비 및 보안 요건
- 선불충전금의 100% 이상 별도 관리 (신탁·예치·지급보증보험)
- 5인 이상 전산 전문인력

위반 시: **3년 이하 징역 또는 2천만 원 이하 벌금** (제49조)

### 등록 면제 조건 (제28조 제3항)

다음 각 호의 어느 하나에 해당하는 경우 등록 의무가 면제된다.

**제1호 (단일 가맹점 사용)**
> 하나의 가맹점(가맹점의 사업주가 동일한 경우로 한정한다)에서만 사용되는 선불전자지급수단을 발행하는 자

- "하나의 가맹점"은 물리적 점포 하나가 아니라, **동일한 사업주가 운영하는 모든 점포·채널을 하나**로 간주
- 즉, 자사 앱·웹·매장 등 여러 채널이 있어도 사업주가 동일하면 "하나의 가맹점"에 해당

**제2호 (소액 발행 면제)**
> 선불전자지급수단의 발행잔액이 **30억 원 미만**이고 연간 총발행잔액이 **500억 원 미만**인 자

- 발행잔액 30억 원 AND 연간 500억 원, 양쪽 기준을 모두 충족해야 면제
- 2021년 머지포인트 사태 이후 2024년 9월 시행된 개정법으로 규제 강화

**제3호 (무상 포인트)**
> 이용자가 미리 직접 대가를 지불하지 아니한 선불전자지급수단으로서 상환보증보험 등에 가입한 경우

- 이벤트 포인트, 리워드 포인트 등 이용자가 직접 구매하지 않은 경우

### 중요: 면제 조건 논의 이전에 먼저 따져야 할 것

> **전제 조건**: 면제 조건을 적용하려면 먼저 해당 수단이 "선불전자지급수단"에 해당해야 한다.
>
> 견적코치 분석권은 **애초에 선불전자지급수단 정의에 해당하지 않으므로**, 면제 조건을 논의할 필요조차 없다.

## 3. 견적코치 분석권이 선불전자지급수단이 아닌 이유 {#3}

### 견적코치 1회 분석권의 특성

| 특성 | 견적코치 분석권 | 선불전자지급수단 |
|------|---------------|----------------|
| **사용처** | 견적코치 플랫폼 내에서만 사용 가능 | 발행인 외 제3자 가맹점에서도 사용 가능 |
| **금전적 가치 이전** | 타인에게 양도·이전 불가 | 이전 가능한 금전적 가치 |
| **충전 구조** | 없음 (건별 구매) | 충전 후 잔액 유지 |
| **환불 구조** | 미사용 시 청약철회에 따른 환불 (전자상거래법 적용) | 법정 환급 의무 (전자금융거래법 적용) |
| **가맹점 네트워크** | 없음 | 다수의 제3자 가맹점 계약 |
| **본질** | AI 분석 서비스 1회 이용 대가 | 통화 대체 수단 |

### 요건별 불충족 분석

**① 이전가능한 금전적 가치 — 미충족**

견적코치 분석권은 특정 서비스(AI 견적 분석)에 대한 **접근 권한(access right)** 이다. 1회 분석권을 구매했다고 해서 그 권한을 타인에게 팔거나, 현금화하거나, 다른 서비스 결제에 사용할 수 없다. 이는 영화관 티켓, 헬스장 1일 이용권, 놀이공원 1일 입장권과 동일한 법적 성격으로, 모두 "서비스 이용 대가"이지 "금전적 가치의 전자적 저장"이 아니다.

**② 제3자 가맹점 사용 가능성 — 미충족 (핵심)**

분석권은 오직 견적코치(smalltablekorea)가 운영하는 서비스에서만 사용된다. 제3자와의 가맹점 계약이 존재하지 않으며, 분석권으로 다른 쇼핑몰, 식당, 콘텐츠 서비스 등에서 결제할 방법 자체가 없다.

금융위원회의 일관된 해석에 따르면:
> "발행인인 A사만의 재화 또는 용역을 구입하는 데 사용할 수 있다면 선불전자지급수단에 해당하지 않는다."

**③ 충전·잔액 구조 부재**

선불전자지급수단의 일반적 특성은 "먼저 돈을 충전하여 잔액이 쌓이고, 나중에 여러 곳에서 차감하여 사용"하는 구조다. 견적코치 분석권은:
- 잔액이 쌓이는 구조가 아님 (분석권 N회 = 개별 서비스 N개)
- 충전 후 미사용 잔액이 발생하는 구조가 아님
- 서비스 미이행 시 전자상거래법상 청약철회로 처리

### 법적 성격의 정확한 분류

견적코치 분석권의 법적 성격:

1. **민법상**: 용역(서비스) 공급 계약에서의 이용 대가 선급 (민법 제664조 이하 도급 계약 관련)
2. **전자상거래법상**: 디지털 콘텐츠 또는 온라인 서비스의 이용권 판매 (전자상거래등에서의소비자보호에관한법률 적용)
3. **부가가치세법상**: 용역 공급 대가 (세금계산서 또는 현금영수증 발급 대상)
4. **전자금융거래법상**: **해당 없음** — 선불전자지급수단의 3대 요건 미충족

### 유사 사례와의 비교

| 서비스 유형 | 법적 성격 | 선불전자지급수단 해당 여부 |
|------------|----------|------------------------|
| 카카오페이 충전잔액 | 전자적으로 저장된 금전적 가치, 제3자 가맹점 사용 가능 | **해당** |
| 스타벅스 모바일 쿠폰 (자사 직영점 전용) | 단일 사업주 가맹점 전용 | **면제 (제28조 제3항 제1호 적용)** |
| CGV 영화 관람권 | CGV라는 단일 사업자 서비스 이용권 | **해당 없음** |
| 헬스장 1개월 이용권 | 특정 서비스 이용 대가 | **해당 없음** |
| Adobe Stock 크레딧 팩 | 어도비 자사 플랫폼 내 전용 사용 | **해당 없음** |
| 견적코치 1회 분석권 | 자사 AI 서비스 1회 이용 대가 | **해당 없음** |

## 4. 금융위원회 유권해석 및 비조치의견서 사례 {#4}

### 사례 1: 서비스 포인트 및 게임 포인트 판단 (2024년 3월 27일)

**출처**: 금융위원회·금융감독원 비조치의견서 (CaseNote 등재)

**핵심 내용**:
- A사의 서비스 포인트가 **다른 사업자의 선불전자지급수단으로 전환**하거나 **모바일 상품권 구매**에 사용될 수 있다면 → 선불전자지급수단에 **해당**
- A사의 게임 포인트도 동일한 기준 적용
- **판단 핵심**: 제3자의 재화·용역 구입에 사용 가능한지 여부

**견적코치와의 비교**: 분석권은 타사 서비스로 전환하거나 타사 상품권 구매에 사용하는 것이 **구조적으로 불가능**하므로, 이 사례의 "해당" 요인이 존재하지 않는다.

### 사례 2: 선불전자지급수단 발행자 판단 기준 (2024년 6월 10일)

**출처**: 금융위원회·금융감독원 비조치의견서 (CaseNote 등재)

**핵심 내용**:
- 모바일상품권 교환권이 특정 브랜드의 상품을 교환할 수 있는 경우 → 선불전자지급수단 **해당**
- "선불결제를 위한 증표를 발행·관리하고 **가맹점에 대한 결제 업무를 수행**하는" 자가 등록 의무를 지는 실질적 발행자
- 단순히 **판매·유통만** 하는 경우는 선불업자 등록 불필요

**견적코치와의 비교**: 견적코치는 자사 서비스의 이용권을 판매하는 것이며, 제3자 가맹점에 대한 결제 업무를 전혀 수행하지 않는다.

### 사례 3: 업장별 포인트 (세차장 플랫폼 사례)

**출처**: 금융감독원 비조치의견서 (CaseNote 등재, 2024)

**핵심 내용**:
- 세차장 플랫폼이 각 업장별로 포인트를 발행하되, **해당 업장에서만** 사용 가능한 경우
- → 전자금융거래법 제28조 제3항 제1호 적용: "하나의 가맹점에서만 사용" → **등록 면제**
- 실질적인 발행·관리 주체를 기준으로 판단해야 함

**견적코치와의 비교**: 설령 분석권이 선불전자지급수단에 해당한다고 봐도(이것이 잘못된 가정이지만), 견적코치라는 단일 사업주가 운영하는 단일 서비스에서만 사용되므로 등록 면제 대상이다.

### 사례 4: 음식점 가맹본사 선불카드 (업종 1개 사용)

**출처**: 금융규제·법령해석포털 법령해석 회신 (stNo=11, lawreqIdx=1395)

**핵심 내용**:
- 가맹본사가 발행한 선불카드로 **음료 + MD 제품**을 구매할 수 있어도, 주된 영업이 동일 업종(음식점업)이면 "업종 1개"로 봄
- 이 경우 선불업 등록 대상이 아닌 것으로 해석

**견적코치와의 비교**: 분석권은 "인테리어 견적 AI 분석"이라는 단일 서비스에만 사용되므로, 이 사례보다 훨씬 명확히 등록 대상이 아니다.

### 금융당국의 일관된 해석 원칙

수집된 유권해석들에서 공통적으로 확인되는 원칙:

> 1. **제3자성 요건이 핵심**: 발행인 자신의 재화·용역 구매에만 사용된다면 선불전자지급수단이 아니다.
> 2. **실질적 판단**: 명칭(크레딧, 포인트, 토큰 등)보다 실질적인 사용 구조를 본다.
> 3. **단일 사업주**: 동일 사업주가 운영하는 복수 채널은 "하나의 가맹점"으로 본다.

## 5. PG사 심사에서 문제가 되는 용어 목록 (블랙리스트) {#5}

아래 용어들은 PG사(특히 토스페이먼츠, KCP, 이니시스 등)의 자동 심사 시스템 또는 심사 담당자에게 "선불전자지급수단 의심" 또는 "충전 서비스 의심"을 유발한다.

### 직접 사용 금지 용어

| 용어 | 금지 이유 | PG 위험도 |
|------|-----------|----------|
| **크레딧 (Credit)** | 카드사 "신용"을 연상시키거나, 충전식 가상화폐 개념 연상 | ★★★★★ |
| **포인트 (Point)** | 멤버십 포인트·충전 포인트와 혼동 | ★★★★☆ |
| **토큰 (Token)** | 암호화폐·디지털자산 연상, 금융당국 민감 대응 | ★★★★★ |

| **환전 (Exchange)** | 화폐 교환 기능 연상 | ★★★★★ |
| **발행 (Issue)** | 선불 수단 발행 개념 연상 | ★★☆☆☆ |
| **충전권** | 충전 직접 명시 | ★★★★★ |

### 왜 "크레딧"이 문제가 되었나

토스페이먼츠의 가맹점 심사는 상품명, 서비스 설명, 결제창 UI 등에서 **키워드 기반 스크리닝**을 수행한다. "크레딧"은:

1. 카드사 신용(credit) 연상 → 규제 확인 필요
2. AI 서비스 크레딧(OpenAI API Credits 등) 연상 → 금전적 가치로 오해
3. "충전형 게임 크레딧" 연상 → 1일 결제한도 5만원 규제 적용 가능 업종과 혼동

이는 서비스의 법적 성격과 무관하게 **심사관의 검토 부담을 높이고 심사 지연·거절로 이어질 수 있다**.

**ChatGPT/OpenAI API (한국 서비스)**

- 제3자 가맹점 없음 → 선불전자지급수단 해당 없음

**국내 다수의 AI SaaS (예: 특허 분석, 법률 AI, HR AI 등)**
- "분석 건수", "이용권", "패스" 등의 용어 사용
- 자사 플랫폼 내에서만 사용 → 선불전자지급수단 해당 없음

### 결론적 유추

위 서비스들의 공통점:
1. 발행사 자신의 단일 서비스 내에서만 사용 가능
2. 제3자 가맹점 네트워크 없음
3. 이전·양도 불가 구조
4. 전자금융업 등록 없이 일반 PG사를 통해 결제 처리

이들이 정상적으로 PG 계약을 유지하고 있다는 사실은 "자사 서비스 전용 이용권"이 선불전자지급수단이 아님을 실무적으로 입증한다.

## 8. 결론 및 토스페이먼츠 심사 대응 요령 {#8}

### 법적 결론

**견적코치 1회 분석권은 전자금융거래법상 선불전자지급수단이 아니다.**

근거:
1. 전자금융거래법 제2조 제14호의 "발행인 외의 제3자로부터 재화·용역을 구입하는데 사용"이라는 핵심 요건을 충족하지 못함
2. 금전적 가치의 이전 구조가 없으며, 단순히 자사 AI 서비스의 1회 이용 대가
3. 제3자 가맹점 계약 없음, 충전 구조 없음, 이전·양도 불가
4. 금융위원회의 일관된 유권해석에 따르면 자사 서비스 전용 이용권은 선불전자지급수단에 해당하지 않음

따라서:
- 전자금융업(선불전자지급수단 발행 및 관리업) **등록 의무 없음**
- 선불충전금 별도 관리 의무(100% 보호) **해당 없음**
- 금융위원회 관할 규제 **해당 없음**

**적용 가능한 법률**: 전자상거래등에서의소비자보호에관한법률(환불 정책), 개인정보보호법, 부가가치세법(용역 공급)

### 토스페이먼츠 심사 대응 방법

**즉시 조치사항**:

1. **용어 교체**: 서비스 내 "크레딧" → "분석권" 또는 "1회 이용권"으로 전면 교체
2. **결제창 상품명 변경**: `크레딧 충전` → `AI 분석 이용권 구매` 또는 `1회 분석권 구매`
3. **이용약관 명시**: 분석권이 자사 서비스에서만 사용되며 제3자에게 이전 불가임을 약관에 명시

**심사관 문의 시 답변 포인트**:

> "본 서비스에서 판매하는 '분석권'은 전자금융거래법상 선불전자지급수단이 아닙니다. 분석권은 ① 견적코치 플랫폼 내에서만 사용 가능하고, ② 제3자 가맹점 사용이 불가능하며, ③ 타인에게 이전·양도할 수 없는 서비스 접근 권한입니다. 영화관 티켓이나 헬스장 이용권과 동일한 법적 성격으로, 전자금융거래법 제2조 제14호의 선불전자지급수단 정의 요건 중 '발행인 외의 제3자로부터 재화·용역을 구입하는데 사용' 요건을 충족하지 않습니다."

**필요 시 제출 가능한 근거 자료**:

- 이용약관: 분석권의 사용 범위(자사 전용), 양도 금지 조항 명시
- 서비스 소개 페이지: 분석권이 자사 AI 서비스 1회 이용권임을 설명
- (필요시) 금융위원회 비조치의견서 신청 가능 (better.fsc.go.kr)

### 향후 서비스 설계 시 주의사항

아래 조건을 유지하는 한 전자금융업 등록 의무는 발생하지 않는다:

- [x] 분석권을 견적코치 서비스 내에서만 사용 가능하도록 유지
- [x] 분석권의 타인 양도·이전 기능을 구현하지 않음
- [x] 분석권을 다른 서비스나 제3자 가맹점의 결제 수단으로 사용하는 기능을 구현하지 않음
- [x] 분석권의 현금화(환전) 기능을 구현하지 않음

아래 경우에는 법률 전문가 재검토 필요:

- [ ] 다른 파트너사(제3자)가 분석권을 함께 발행하거나, 분석권으로 결제를 받는 구조 도입 시
- [ ] 분석권을 B2B 대리점이 구매 후 최종 이용자에게 재판매하는 구조 도입 시
- [ ] 분석권 발행 잔액이 30억 원을 초과하기 시작하는 성장 단계
